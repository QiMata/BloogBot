## Identity

**Zul'Gurub is the 20-man Troll raid** in Stranglethorn Vale, **the only short-lockout raid (3 days)** in vanilla, **no attunement required**, and the source of [Zandalar Tribe](#) rep + the **Bloodvine** crafting gear path. ZG sits between MC and BWL in difficulty + gear quality; many guilds run ZG **2-3x per week** for the rep + farmable epics + Hakkari Coins for class-specific necklaces.

| Feature | Value |
|---|---|
| Raid size | **20-player** (smaller than 40-mans; faster setup) |
| Level required | 60 |
| Attunement | **None** (anyone lvl 60 can enter) |
| Lockout | **3 days** (shortest in vanilla — most raids are 5/7 days) |
| Full clear time | 1.5-2.5 hours (typical guild) |
| Boss count | 6 mandatory + 4-5 optional + Edge of Madness mini-boss event |
| Location | Stranglethorn Vale, north of Booty Bay (Mudsprocket area) `[verify exact entrance pass 3]` |

## Boss list — 5 High Priests + Hakkar + optional bosses

ZG's main encounter is **5 High Priest/Priestess bosses + Hakkar the Soulflayer** as the final boss. Each Priest **empowers Hakkar with ~50,000 HP per surviving priest** if not killed first — so **all 5 must die before Hakkar** for the optimal kill.

| # | Boss | Type | Aspect / mechanic |
|---|---|---|---|
| 1 | **High Priestess Jeklik** | Mandatory High Priest | **Bat aspect** — wing flap + Bat Form + Sonic Burst |
| 2 | **High Priest Venoxis** | Mandatory High Priest | **Snake aspect** — venom debuffs + Snake Form transformation |
| 3 | **High Priestess Mar'li** | Mandatory High Priest | **Spider aspect** — Spider Form + Web Spray + Cocoon |
| 4 | **High Priest Thekal** | Mandatory High Priest | **Tiger aspect** — **3-headed boss with adds Zath + Lor'Khan** that **must die simultaneously** (within ~5 seconds) or revive |
| 5 | **High Priestess Arlokk** | Mandatory High Priest | **Panther aspect** — Vanish + Stealth panther adds spawn |
| 6 | **Bloodlord Mandokir** (optional) | Drako-mounted boss | **Threatening Gaze** mechanic: random raid member must hit zero auto-attacks during stare or take fatal damage; rides **Ohgan** dino-mount; Mandokir respawns on death (~30s) — must coordinate burns |
| 7 | **Edge of Madness** (mini-boss event, optional) | RNG mini-boss | One of 4 mini-bosses spawns per ZG instance (rotation): **Gri'lek** (lightning), **Hazza'rah** (sand-based), **Renataki** (rogue-like), **Wushoolay** (storm). Each drops unique trinket (e.g., Tooth of Eranikus). |
| 8 | **Jin'do the Hexxer** (optional) | Hakkari priest | **Hexes raid members** + **raises Hakkari undead chains** during fight; mind-controls + DoT |
| 9 | **Gahz'ranka** (optional, fishing-summoned) | Aquatic boss | Summoned via **Mudskunk Lure** (Cooking + Fishing requirement); drops Tigule and Foror's Strawberry Ice Cream + the **Tigerseye Brooch** trinket |
| **10** | **Hakkar the Soulflayer** | **FINAL BOSS** | **Aspect of Loa** (-50% healing on raid for 30s); **Blood Siphon** (life-drain from raid); **Cause Insanity** (reverse-mind-control); **Aspect of Sand / Aspect of Water** various life/mana drains. **Each surviving High Priest empowers Hakkar with +50000 HP**. |

## Boss order strategy

Standard guild path:

1. Run path: enter ZG → **Jeklik** (left side) → **Venoxis** (left/snake corridor) → **Mar'li** (north spider sanctum) → **Thekal** (tiger sanctum) → **Mandokir** (optional but recommended for loot) → **Arlokk** (panther sanctum) → **Edge of Madness** (mini-boss spawn area) → **Jin'do** (optional) → **Hakkar** (final pit).

2. Kill order ensures Hakkar weakened: **all 5 priests dead → Hakkar at base HP**.

**Engine optimization**: Plan the kill order to maximize loot from optional bosses (Mandokir, Edge of Madness, Jin'do, Gahz'ranka) since the 3-day lockout means more frequent runs are possible.

## Key boss mechanics — engine-relevant

- **Mandokir Threatening Gaze**: Random raid member targeted; **must hit zero white attacks** (auto-attack swings) during stare or **die instantly**. Tactic: stop melee swings + sit (stand still). Engine alerts: when stared, force "stop attacking" for ~3 seconds.
- **Mandokir respawn**: Boss respawns ~30 seconds after death; raid must time burn correctly to defeat the second cycle.
- **Thekal 3-add simultaneous death**: Thekal + Zath + Lor'Khan must die within ~5 seconds of each other. Engine plans 3 tank assignments + 3 DPS groups + sync kill timer.
- **Arlokk Vanish + Panther summons**: Boss vanishes ~30s; raid AoEs panthers; engine plans target-of-target tracking when Arlokk re-emerges.
- **Hakkar Aspect of Loa**: Raid-wide -50% healing for 30s — burn DPS phase. Heal cooldowns saved for between Aspect cycles.
- **Hakkar Blood Siphon**: Life-drain from raid — healer burst. Some guilds class-specific Resistance gear (Shadow Resist for Hakkar's shadow).
- **Edge of Madness**: One mini-boss per ZG instance. Engine notes which spawn type (Gri'lek/Hazza'rah/Renataki/Wushoolay) for trinket-loot prediction.
- **High Priest empowerment of Hakkar**: Engine warns if Hakkar engaged with surviving priests (50k HP per priest = significant DPS check).

## Zandalar Tribe rep grinding via ZG

**Zandalar Tribe** is the ZG-tied faction. **Bijou turn-ins** (drop from bosses) → 50-100 rep per Bijou. **Coins / Hakkari runes** turn-ins → 25-50 rep each. Most ZG-running guilds reach **Honored within 4-6 weeks** and **Exalted in 8-12 weeks** of weekly ZG attendance.

Rep tiers + rewards:

| Tier | Reward |
|---|---|
| Friendly | ZG tabard purchase + minor recipes |
| **Honored** | Spirit/Sheen/Swiftness of Zanza buff vendor (see [world-buffs.md](../systems/world-buffs.md)); Bloodvine Tailoring patterns |
| **Revered** | Better caster/healer accessories; reduce Coin turn-in cost for class neck |
| **Exalted** | **Insignia of the Alliance/Horde**-equivalent ZG trinkets; library of late-vanilla recipes |

## Decision-Engine Rules

- **id:** `raid.zg.lockout-3-day` — IF `RaidLastZG.Date < (now - 3 days)` THEN raid eligible. Priority **decision-request**.
- **id:** `raid.zg.priest-kill-order` — IF `Hakkar.NotEngaged && SurvivingHighPriests.Count > 0` THEN engine plans priest kills before Hakkar. Priority **999** (combat-time strategy).
- **id:** `raid.zg.mandokir-stare-stop` — IF `Player.IsStaredBy(Mandokir)` THEN stop white attacks for 3 seconds. Priority **999** (combat-time).
- **id:** `raid.zg.thekal-3-add-sync` — IF `Thekal.PhaseSimultaneousKill` THEN engine signals all 3 tank groups for sync kill. Priority **950** (combat-time).
- **id:** `raid.zg.bijou-turn-in` — IF `Items.Contains(Bijou) && Reputation[ZandalarTribe] < Exalted && InMudsprocket` THEN turn in for rep. Priority **400** (background).
- **id:** `raid.zg.coin-class-neck` — IF `Items.HasAllHakkariCoinsForClass && InMudsprocket` THEN turn in 9 Coins for class neck. Priority **600** (gear improvement, one-time).
- **id:** `raid.zg.hakkar-aspect-of-loa` — IF `Hakkar.AspectOfLoa.IsActive` THEN healers save cooldowns; raid burns DPS phase. Priority **900** (combat-time).
