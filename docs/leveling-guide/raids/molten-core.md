## Identity

**Molten Core is the first 40-man raid in vanilla** (lvl-60, accessed via [Lothos Riftwaker MC attune](../attunements/molten-core.md)) and the **most-run raid in 1.12.1** by total /played hours. 10 bosses in linear sequence ending with Ragnaros. **T1 Battlegear of Might / Lawbringer / Cenarion / Earthfury / Vestments of the Devout / Arcanist Regalia / Felheart / Giantstalker / Nightslayer** sets all source from MC bosses, plus the **Bindings of the Windseeker** (Thunderfury legendary mat) from Garr + Geddon.

| Feature | Value |
|---|---|
| Raid size | 40-player |
| Level required | 60 |
| Attunement | [Lothos Riftwaker → Core Fragment](../attunements/molten-core.md) |
| Quintessence requirement | ≥7 raid members with Aqual or Eternal Quintessence (see [hydraxian-waterlords.md](../reputations/hydraxian-waterlords.md)) |
| Lockout | 7 days (weekly reset Tuesday) |
| Full clear time | 4-6 hours (well-geared guild); 6-8 hours (progression) |
| Boss count | 10 |

## Boss list and order

Linear progression — bosses positioned along the Core's lava paths. Skipping is rare; full clears are standard.

| # | Boss | Mechanic highlights | Notable drops |
|---|---|---|---|
| 1 | **Lucifron** | Curse-of-Lucifron debuff (curse-removal), Impending Doom, Lucifron's Curse | Judgement Hood (Pal T1), Felheart Slippers (Warlock T1), other T1 |
| 2 | **Magmadar** | **Frenzy** (Tranq Shot dispel), **Panic** (large fear AoE — Fear Ward / Tremor Totem mandatory), Conflagrate, lava pools | T1 leg pieces (Lightforge Legplates Pal, Necropile Pants Warlock, etc.) |
| 3 | **Gehennas** | Curse of Gehennas, Rain of Fire, Gehennas's Curse (mana cost increase) | T1 gloves (Pal/Shaman/Rogue) |
| 4 | **Garr** | **8 Firesworn adds** that **explode on death** (must die simultaneously OR pulled separately and tanked); Magma Shackles | T1 head pieces all classes; **Bindings of the Windseeker** (left half, ~5%) |
| 5 | **Baron Geddon** | **Living Bomb** (random raid member; runs out of group to detonate); Inferno (raid AoE); Ignite Mana (mana burn) | T1 belts/gloves (Pal Lawbringer Pauldrons / Shaman Stonerender Gauntlets); **Bindings of the Windseeker** (right half, ~5%) |
| 6 | **Shazzrah** | **Blink-teleport** to random raid members; Arcane bursts; Counterspell; Magic Grounding | T1 shoulders + caster robes (Arcanist Regalia / Necropile Vestments) |
| 7 | **Sulfuron Harbinger** | 5 **Flamewaker Priest adds** (heal Sulfuron — must be disrupted/silenced); Demoralizing Shout | **Sulfuron Ingot** (100% drop — used for Sulfuron Hammer recipe BS / Sulfuras legendary reagent) + T1 pieces |
| 8 | **Golemagg the Incinerator** | **2 Core Hounds** (Heat of Ragnaros pets — must die within ~10s of each other or revive); Lava Burst; Magma Splash | T1 ranged + boots; class-specific drops |
| 9 | **Majordomo Executus** | **MUST be summoned via Quintessence-doused 7 runes around Ragnaros's pit** (mandatory mechanic); 4 Honor Guard adds (mana drain — Mages disrupt); becomes **friendly post-fight** (no loot fight; loot from his cache after) | **Cache of the Firelord** drops **Ancient Petrified Leaf** (Hunter Rhok'delar) OR **Eye of Divinity** (Priest Benediction) — 50/50 split per kill |
| 10 | **Ragnaros** | **Submerge phase at 100%/50%** (~30s underwater between phases); **Sons of Flame** spawn during submerge (must die before Ragnaros re-emerges or wipe); Knockback (lava-pit deaths); Magma Burst | **Eye of Sulfuras** (100% first kill — Sulfuras legendary reagent); **Bindings of the Windseeker** does NOT drop here (only Garr/Geddon); various T1 BiS pieces; **Cloak of the Shrouded Mists** (caster) |

**Engine planning rule**: Engine plans full 10-boss clears per weekly reset; suspending early bosses for "skipping" is generally suboptimal (most bosses are required for T1 piece distribution).

## Pre-raid prep checklist

Engine should ensure each raid attendee has:

- [ ] **MC attune** completed (Attunement to the Core spell from Lothos)
- [ ] **Aqual or Eternal Quintessence** in bags (≥7 raid members for Majordomo douse — see [hydraxian-waterlords.md](../reputations/hydraxian-waterlords.md))
- [ ] **Fire Resistance gear** (~150-200 unbuffed FR for melee/tanks; some classes can skip — Paladins use Resistance Aura)
- [ ] **World buffs** (Rallying Cry / Songflower / DM Tribute / Zanza — see [world-buffs.md](../systems/world-buffs.md))
- [ ] **Consumables stack** (flasks, elixirs, jujus, food, healing/mana pots, runes — see [consumables.md](../systems/consumables.md))
- [ ] **Class-specific reagents** (Paladin Symbol of Kings for Greater Blessings, Warlock Soul Shards, Mage Conjured Water — engine should verify pre-pull)

## Notable raid mechanics — engine-relevant

- **Magmadar's Panic fear**: AoE fear, 5-second duration. **Fear Ward (Dwarf Priest only)** applied to MT pre-pull is mandatory for Alliance. Horde uses **Tremor Totem** (Shaman) on a 60-sec rotation.
- **Tranquilizing Shot (Hunter)** mandatory for Magmadar's Frenzy. **Engine rule**: at least 2 Hunters in raid, 1 dedicated to dispel rotation.
- **Garr Firesworn split-tank**: 8 adds + Garr; raid splits into multiple groups to tank Firesworn separately, kill them in sequence (or pull all then AoE down). Banish (Warlock) on Firesworn is the optimal control.
- **Baron Geddon Living Bomb**: random target gets 8-second debuff that detonates AoE on expiry. **Bomber must run out of group** to ≥10 yards. Engine warns target via in-raid alert.
- **Sulfuron Flamewaker Priests**: 5 priests heal Sulfuron. Disruption priority: Mages Counterspell + Warriors Pummel + Rogues Kick + Priest Silence (Shadow). Engine plans interrupt rotation.
- **Golemagg Core Hounds**: 2 hounds die simultaneously or revive. Each hound on a separate tank, raid stacks DPS to ensure synchronized kills.
- **Majordomo Quintessence ritual**: 7 runes around Ragnaros's pit doused via Quintessence (per-character single-cast). Engine verifies ≥7 attendees with Quintessence pre-event.
- **Ragnaros Sons of Flame**: every ~3 minutes Ragnaros submerges, spawning 6-8 Sons of Flame (lvl 63 elites that 1-shot melee). Raid AoE-burns Sons before Ragnaros re-emerges. Failure = 1-2 raid members die, but recoverable.

## Decision-Engine Rules

- **id:** `raid.mc.attune-required` — IF `RaidScheduledMC && !Attunements.Contains(MoltenCore)` THEN raid action suppressed. Priority **999**.
- **id:** `raid.mc.quintessence-coverage` — IF `RaidScheduledMC && Sum(RaidMembers.HasQuintessence) < 7` THEN warn raid leader; engine flags coverage gap. Priority **999**.
- **id:** `raid.mc.fire-resist-target` — IF `Player.IsRaidMC && Class IN {Warrior, Druid, Paladin (tank)} && FireResistance < 150` THEN engine suggests FR gear acquisition. Priority **600**.
- **id:** `raid.mc.world-buff-window` — IF `RaidScheduledMC && Raid.PullTime within 90 minutes && WorldBuffsActive < 3` THEN initiate world-buff prep window. See [world-buffs.md](../systems/world-buffs.md). Priority **750**.
- **id:** `raid.mc.bindings-priority` — IF `Account.LegendaryThunderfury IN PlannedItems && BindingsCount < 2` THEN engine flags Garr + Geddon as priority-loot bosses. Priority **decision-request** (raid loot policy decision).
- **id:** `raid.mc.cache-priority-class` — IF `RaidContains(MultipleHunters) AND RaidContains(MultiplePriests)` THEN engine plans rotation: Cache loot priority for player who needs item most for class quest progression. Priority **decision-request**.
