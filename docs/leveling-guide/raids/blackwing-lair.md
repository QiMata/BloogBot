## Identity

**Blackwing Lair is the second 40-man raid** (third counting Onyxia) and is **substantially harder than MC + Onyxia combined**. 8 progression-defining bosses including the **Razorgore mind-control / egg-detonate** progression wall, **Vaelastrasz 5-min hard enrage** burn-down, **Chromaggus 5-debuff cycling** dispel test, and **Nefarian class-call** finale. Drops **T2 non-helm pieces all 9 classes** plus **Elementium Ore** for Thunderfury crafting.

| Feature | Value |
|---|---|
| Raid size | 40-player |
| Level required | 60 |
| Attunement | [Mark of Drakkisath](../attunements/blackwing-lair.md) (Blackhand's Command + UBRS Drakkisath orb touch) |
| Lockout | 7 days (weekly reset Tuesday) |
| Full clear time | 4-6 hours (geared guild); 8-12 hours (progression) |
| Boss count | 8 |
| Location | Inside Blackrock Spire (BWL portion); accessed via Orb of Command |

## Boss list and order

Linear progression — bosses arranged in a top-down path from Vaelastrasz's chamber (entry) through the dragon halls to Nefarian.

| # | Boss | Mechanic highlights | Notable drops |
|---|---|---|---|
| 1 | **Razorgore the Untamed** | **Mind-control mechanic**: 1-2 Mages take control of Razorgore via summoned orbs and use his Destroy ability to break **30+ Black Dragon Eggs** in the chamber while raid kills adds (Death Knight Drak, Grethok the Controller, eggs themselves). Phase 2: free Razorgore + raid burn. **Progression wall** for many guilds. | T2 hands all 9 classes (Gauntlets of Wrath / Judgement Gauntlets / etc.) `[verify pass 3]` |
| 2 | **Vaelastrasz the Corrupt** | **5-minute hard enrage**: must die before; **Burning Adrenaline** debuff (random raid member, 30s detonation similar to Geddon Living Bomb but more lethal); Fire Nova; **friendly NPC turned hostile** narrative | T2 legs all 9 classes (Legplates of Wrath / etc.); Drake Talon Cleaver (Warrior 2H sword), Tooth of Gruul (caster) `[verify pass 3]` |
| 3 | **Broodlord Lashlayer** | **Mortal Strike** (50% healing reduction on tank); **Knock Away** (aggro reset); **Blast Wave** (raid AoE); **Suppression debuff** (random raid member, magic dispel) | T2 wrist all 9 classes; Maladath, Runed Blade of the Black Flight (Warrior/Rogue 1H sword) |
| 4 | **Firemaw** (drake trio member 1) | **Wing Buffet** (stack-debuff on melee, ~20 stacks → 1-shot melee — kite mechanic); **Shadow Flame** (frontal cone fire); rotating tank assignments | T2 belts (mixed across drakes) |
| 5 | **Ebonroc** (drake 2) | **Mortal Strike**; **Wing Buffet** stack; **Healthsteal** (heals Ebonroc as he damages tank); rotating tanks | T2 belts/boots |
| 6 | **Flamegor** (drake 3) | **Frenzy** (Tranq Shot dispel); **Wing Buffet**; rotating tanks | T2 boots all 9 classes |
| 7 | **Chromaggus** | **Brood Affliction**: 5 chromatic debuffs (Black/Bronze/Blue/Red/Green) cycling on raid; **only 4-5 raid healers can dispel** all 5 types (Mages decurse Bronze, Druids Decurse, Priests Dispel, Pal Cleanse — coverage matters); **5-min hard enrage**; **Frenzy** (Tranq Shot); **2 Breath weapons** randomized per kill (Time Lapse, Incinerate, Frost Burn, Corrosive Acid, Ignite Flesh) — different mechanic each pull | T2 shoulders all 9 classes; Chromatically Tempered Sword |
| 8 | **Nefarian** | **2-phase encounter**: (P1, 100→25%): Drakonid adds; tank Nefarian with rotating threat; (P2, 25→0%): **Class Call** mechanic — every ~30s a class-wide ability fires (e.g., "All Mages will be Polymorphed", "All Druids will be Wild Polymorphs", "All Warriors will be Berserker Stance with no rage cost") — raid must adapt instantly. **Bone Construct adds** spawn during P2. Chrome eggs hatch. | T2 chests all 9 classes (Breastplate of Wrath / etc.); **Head of Nefarian** (100% drop — turns in for **Rallying Cry of the Dragonslayer** world buff, same mechanic as Onyxia head); class-specific epics |

## Notable raid mechanics — engine-relevant

- **Razorgore mind-control progression**: 1-2 Mages designated as **MC controllers**. Engine pre-assigns Mage role + verifies orb interactions in chamber.
- **Vaelastrasz Burning Adrenaline run-out**: Random raid member gets 30s timer to detonate. Must run away from raid + die. Engine alerts target via raid system.
- **Vaelastrasz 5-minute hard enrage**: Stricter than Chromaggus's 5-min enrage; raid DPS check. Engine times burn rate.
- **Broodlord Mortal Strike**: Tank takes 50% reduced healing during MS uptime; healer rotation must compensate.
- **Drake trio Wing Buffet stack**: Each Wing Buffet adds a stack on tank/melee. **Tank rotation every ~6-8 stacks** to prevent 1-shots. Engine plans 2-3 tank rotation per drake.
- **Chromaggus Brood Affliction**: 5 random debuffs spread across raid every ~30s. Coverage requires diverse dispel composition. Engine rule: ensure raid has Mage+Druid (curses), Priest+Pal (magic+disease), Druid (poison) — full 4-coverage.
- **Nefarian Class Call**: Every ~30s during P2, a class-wide debuff fires. Notable calls:
  - "Mages will Polymorph": all Mages auto-cast Polymorph on random raid → CC management
  - "Druids will be Wild Polymorphed": Druids forced into random forms
  - "Warriors will Berserker Stance": forced stance switch + rage drop
  - "Hunters will be Bestial Wrath'd": pets unleashed
  - "Priests will Mind Control / Levitate": forced casts
  - "Rogues will Vanish": Vanish forced on Rogues
  - "Paladins will Bubble": forced Divine Shield (purgable)
  - "Shamans will go Ghost Wolf": forced form
  - "Warlocks will Demonform / Soul Shard nuke": forced effect
  - Engine plans class-call response per class (e.g., Mages stand on isolated targets to minimize Polymorph chaos).
- **Bone Construct adds** spawn during Nefarian P2 — adds AoE'd by Mages/Warlocks.

- **Head of Nefarian → Rallying Cry world buff** triggers correctly in capital.

## Decision-Engine Rules

- **id:** `raid.bwl.attune-required` — IF `RaidScheduledBWL && !Attunements.Contains(BlackwingLair)` THEN raid action suppressed. Priority **999**.
- **id:** `raid.bwl.razorgore-mc-mage` — IF `Razorgore.IsAlive && PlayerIsMC-Mage && OrbsInRange` THEN initiate Razorgore mind-control orb interaction. Priority **999** (combat-time).
- **id:** `raid.bwl.vael-burn-rate-check` — IF `Vaelastrasz.HP > 50% && Vaelastrasz.TimeAlive > 150s` THEN raid DPS check failed; consider Soulstone-rez engineering / wipe. Priority **999** (combat-time).
- **id:** `raid.bwl.burning-adrenaline-runout` — IF `Player.HasBurningAdrenaline && DistanceToRaid < 10` THEN run out to ≥10y. Priority **999** (combat-time, 30s detonation).
- **id:** `raid.bwl.broodlord-tank-rotation` — IF `Broodlord.MortalStrike.OnTank && TankHealthEffective < 60%` THEN rotate to backup tank. Priority **900** (combat-time).
- **id:** `raid.bwl.drake-wing-buffet-rotation` — IF `Drake.TankWingBuffetStacks >= 6` THEN tank swap. Priority **900**.
- **id:** `raid.bwl.chromaggus-dispel-coverage-check` — IF `RaidComposition.MageCount < 1 OR PriestCount < 2 OR PalCount < 2 OR DruidCount < 1` THEN warn raid leader; engine flags coverage gap. Priority **decision-request**.
- **id:** `raid.bwl.chromaggus-tranq-rotation` — IF `Chromaggus.Frenzy.IsActive && Hunters.TranqShot.OnCooldown == false` THEN dispel. Priority **999**.
- **id:** `raid.bwl.nef-class-call-respond` — IF `Nefarian.P2 && ClassCall.Detected(Player.Class)` THEN execute class-specific response. Priority **950** (combat-time per class).
- **id:** `raid.bwl.head-turnin-priority` — IF `Items.Contains(HeadOfNefarian) && InCapital` THEN turn in for Rallying Cry world buff. Priority **820**.
