## Standard sections per faction file (pass 7 contract)

   - Quests (with one-time XP-equivalent rep value)
   - Mob grinds (per-kill rep at given tier; check for diminishing-returns)
   - Item turn-ins (Scourgestones, Insignia, Bone Fragments, etc.) — rate per hour
2. **Rep gain rate per hour** for each method
3. **Key rewards per tier** (Friendly / Honored / Revered / Exalted)

5. **Mounts unlocked at Exalted** (where applicable)
6. **Decision-Engine Rules**
7. **Snapshot Fields Needed**

## Cross-cutting note: rep diminishing returns

In 1.12.1 there is **no rep-cap-per-day** mechanic for most factions (that's a Cataclysm+ change for some), but several factions do gate turn-ins behind tier thresholds (Argent Dawn Bone Fragments → only Honored+; Cenarion Encrypted Twilight Texts → tier-gated; Thorium Brotherhood Lava Cores → Friendly+).
