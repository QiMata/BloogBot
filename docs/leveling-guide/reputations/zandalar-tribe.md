# Zandalar Tribe — ZG Raid Rep, Yojamba Isle, Bijou + Coin Turn-Ins, Bloodvine Set

Trolls of Zandalar Isle. Hub: **Yojamba Isle** (north Stranglethorn coast). Quartermaster: **Vinchaxa**. **ZG (Zul'Gurub) raid-locked** rep — boss kills + Bijou + Coin turn-ins are the primary sources. Required for: **Bloodvine Set** (Tailoring 3-piece +1% spell crit), **Bloodfang Battlegear** (LW Druid set), Cenarion-aligned crafted gear. Time-to-Exalted: ~30-50 hours (multi-month with 3-day ZG lockout = 2 clears/week). Cross-rep with **Cenarion Circle** (some items rep-locked at both factions).

| Field | Value |

| Time to Exalted | ~30-50 hours (10-20 weekly clears) |

|------------|----------------|

**Decision-engine rule:** Bloodvine 3/3 is **bracket-defining for caster PvP cloth set**. Engine should plan ZG clears for Tailor alts pursuing.

## Reputation Grinding Sources

### ZG Raid Boss Kills (Primary)

ZG raid bosses give Zandalar Tribe rep:

| Boss | Approximate rep gain |
|------|----------------------|
| **Bloodlord Mandokir** | ~150 rep |
| **Hakkar (final)** | ~500 rep |
| **High Priest Thekal** | ~150 rep |
| **High Priest Venoxis** | ~150 rep |
| **High Priest Jeklik** | ~150 rep |
| **High Priestess Mar'li** | ~150 rep |
| **High Priest Arlokk** | ~150 rep |
| **High Priestess Arlokk** *(`[verify pass 3]`)* | (alt) |
| **Mandokir** | (above) |
| **Edge of Madness** sub-bosses | ~75 rep each |
| **Full clear total** | ~1500-2000 rep per week (3-day lockout = 2x/week) |

### Bijou Turn-Ins (Accelerator)

**Bijou** are colored gems that drop from ZG bosses + trash. Turn in 5 of the **same color** at Vinchaxa for rep:

| Color | Drop source |
|-------|-------------|
| **Red Bijou** | Various ZG bosses |
| **Blue Bijou** | Various |
| **Yellow Bijou** | Various |
| **Orange Bijou** | Various |
| **Green Bijou** | Various |

| Turn-in | Rep gain |
|---------|----------|
| **5 same-color Bijou** | ~150-250 rep per turn-in |
| **All 5 colors complete** (Bijou Set Bonus) | Bonus rep + small reward |

**Decision-engine rule:** Bijou turn-ins are the **fastest Zandalar rep accelerator**. Engine should always-loot Bijou drops + batch turn-ins (5 per color).

### Coin Sets (Mid-Game)

**Hakkari Coins** drop from ZG bosses. Turn in **9 of one type** at Vinchaxa:

| Coin type | Drop source |
|-----------|-------------|
| **Bloodlord Coin** | Mandokir-tier bosses |
| **Witchdoctor Coin** | Witch Doctor mobs |
| **Drakkari Coin** | Various |
| (multi-coin sets) | Variants — need 9 of same |

| Turn-in | Rep gain |
|---------|----------|
| **9 same-type Coins** | ~250 rep + Honor Token |

### Mosh'aru Tablet (Cross-Rep)

**Mosh'aru Tablets** (5 needed for ZG Hakkar attune) come from ZF + Sunken Temple. Turning them in to Vinchaxa gives **Zandalar rep** + **AQ40 attune progress** (cross-rep).

**Decision-engine rule:** Mosh'aru Tablets bridge ZF + ST + ZG attune + Zandalar rep simultaneously. Engine should always-loot from ZF Pyramid + ST Avatar of Hakkar.

### Punctured Voodoo Doll (Niche)

Drops from various ZG mobs; ~75 rep per turn-in.

## Decision-Engine Rules

1. **ZG-locked source**: rep is **ZG-only** (Bijou + Coin + boss kills exclusive). Engine should plan ZG weekly attendance.
2. **Bijou priority**: ~150-250 rep per 5-bijou turn-in. Engine should always-loot all 5 colors.
3. **Coin set priority**: 9 same-type coins = ~250 rep + Honor Token. Engine should track per-character coin count.
4. **Mosh'aru Tablet cross-rep**: bridges ZF + ST + ZG attune + Zandalar rep. Engine should always-loot.
5. **Bloodvine set priority**: BiS PvP cloth caster set; engine should plan multi-month chain.
6. **Bloodfang Battlegear priority**: Druid Feral LW set; alternative to Wolfshead.
7. **ZG 3-day lockout**: 2 clears per week. Engine should plan post-Naxx-attune coordination with raid week schedule.
8. **Cross-Cenarion sync**: ZG + AQ + Silithus quests overlap. Engine should batch.
9. **Vinchaxa quartermaster**: Yojamba Isle (north STV); engine should batch trips for turn-ins.
