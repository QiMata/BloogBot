# Cenarion Circle — AQ40 Attune Gate, Silithus Cenarion Hold, Recipe Rewards

Neutral faction representing the druidic Cenarion Circle. Cross-faction (both factions grind same way). **Friendly is the AQ40 attune unlock**; Honored unlocks 1 BoP recipe per profession; Revered unlocks Cenarion Reservist gear set; Exalted is the BiS Cenarion Vestments + Cenarion Battlegear. Rep grinding zones: **Silithus** (Cenarion Hold + Twilight Hammer infiltration) is densest; **Felwood** Emerald Sanctuary + Druid corruption quests; **Desolace** Cenarion Outpost + Maraudon. **Twilight Encrypted Texts** drop in Silithus and turn in for ~20 rep each — bracket-defining accelerator.

| Field | Value |

| Time to Exalted | ~50-100 hours (sustained AQ40 + Silithus farming) |

## Reputation Tier Rewards

### Friendly (AQ40 Attune)

- **AQ40 raid attune chain unlocked** — required to begin AQ40 progression
- Bor Wildmane (Cenarion Hold) sells the **Cenarion Reagent stack** (turn-in for ~75 rep each — basic accelerator)

### Honored (1 BoP Recipe per Profession)

| Profession | Honored recipe |
|------------|----------------|
| **Cooking** | **Runn Tum Tuber Surprise** (+10 Intellect food, 30 min — caster raid food) |
| **Tailoring** | **Cenarion Reservist Robe** (cloth caster chest); **Mooncloth Bracers** |
| **Leatherworking** | **Cenarion Wraps** + **Cenarion Boots** (leather Druid Resto) |
| **Alchemy** | **Cenarion Cleansing Brew** (mid-tier consumable); **Major Healing Potion** (already exists as cross-rep) |
| **Engineering** | **Tranquil Mechanical Yeti** recipe (vanity pet, Cenarion-flavor) |
| **Enchanting** | **Enchant Bracer - Healing Power** (+24 healing); **Enchant Bracer - Mana Regeneration** |

### Revered (Mid-Tier Crafted Gear)

| Profession | Revered recipe |
|------------|----------------|
| **Tailoring** | **Cenarion Vestments** chest (cloth caster — top pre-raid alternative to Mooncloth Vest) |
| **Leatherworking** | **Cenarion Helm** (Druid form-helm); **Cenarion Reservist Boots** |
| **Cooking** | **Cenarion Carrot Cake** (rare food) |
| **Alchemy** | **Restorative Potion** |
| **Enchanting** | **Enchant Cloak - Greater Resistance** (+5 all res); **Enchant Bracer - Spirit** |

### Exalted (BiS Rep-Locked Items)

| Profession | Exalted recipe |
|------------|----------------|
| **Tailoring** | **Cenarion Vestments full set** (BiS pre-raid cloth caster) |
| **Leatherworking** | **Cenarion Battlegear chest + helm** (top tier rep-locked LW gear) |
| **Cooking** | Cenarion-flavor consumables |
| **Alchemy** | **Cenarion Plant Salve** (rare healing alchemy) |
| **All** | **Cenarion-only flavor items** (cosmetics, mounts in TBC) |

**Decision-engine rule:** Exalted Cenarion is a **multi-month grind** (50-100 hours). Engine should plan multi-character coordination across Silithus + AQ20 + AQ40 weekly clears.

## Reputation Grinding Sources

### Silithus (Cenarion Hold) — Primary Grind

Silithus is the **densest rep zone**. Sources:

| Source | Rep gain | Notes |
|--------|----------|-------|
| **Cenarion Hold quests** (10+ available) | ~50-150 rep per quest | Bor Wildmane main hub; complete all on first visit |
| **Twilight Encrypted Tome** | ~20 rep per turn-in | Drops from Twilight's Hammer cultists; turn in at Cenarion Hold; ~25-50 per hour grinding pace |
| **Twilight Cultist Robe set** (3 pieces: Robe, Hood, Cuffs) | ~75 rep per piece | Drop from cultists; turn in for accelerated rep |
| **Twilight Lexicon** (Twilight Lexicon Tome) | ~250 rep per turn-in (rare) | Drop from Twilight Council elite; AQ40 attune component |
| **Anubisath Idol** | ~250 rep per turn-in | AQ20-tier reagent + rep |
| **Silithyst Mound** (1.12 world PvP) | +1 honor (no Cenarion rep) | Not Cenarion-specific |

**Decision-engine rule:** Silithus is **the primary Cenarion grind**. Engine should plan 4-6 hour blocks of Twilight cultist farming + quest turn-ins.

### Felwood (Emerald Sanctuary) — Secondary

Felwood gives Cenarion rep + Timbermaw rep parallel:

| Source | Rep gain |
|--------|----------|
| **Felwood Cenarion quests** (Druid trainer + Arathandris Silversky chains) | ~25-100 rep per quest |
| **Cenarion Tainted Plant Reagent** turn-ins (Grimstreak Bear Spore, Corrupted Songsteel Tracking, etc.) | ~10-25 rep per item |
| **Druid Pristine Hide** quest (turn-in for Druid class chain) | ~250 rep one-time |

### Desolace (Cenarion Outpost) — Tertiary

Cenarion Outpost in Desolace gives Cenarion rep + Maraudon access:

| Source | Rep gain |
|--------|----------|
| **Maraudon Cenarion quests** (Theradric Crystal Carving turn-ins) | ~25-75 rep per turn-in |
| **Cenarion Outpost quests** (Magram/Gelkis centaur kills, kodo round-up — these give Cenarion rep) | ~25-50 rep per quest |

### AQ20 + AQ40 Boss Kills

Every boss kill in AQ20 + AQ40 gives Cenarion rep:

| Raid | Per-clear rep |
|------|---------------|
| **AQ20** | ~500-1000 per full clear (6 bosses) |
| **AQ40** | ~3000-5000 per full clear (9 bosses) |

**Decision-engine rule:** AQ raids are **Cenarion rep accelerator** post-Friendly. Engine should plan weekly AQ20 (3-day lockout = 2x per week) + AQ40 (7-day lockout = 1x per week) for sustained Honored→Exalted progression.

## AQ40 Attune Chain Integration

Cenarion Circle Friendly is **mandatory** for AQ40 entry. The attune chain:

| Step | Action | Source |
|------|--------|--------|
| 1 | Reach Cenarion Circle Friendly via Silithus quests | Silithus questing (~10-20 hours) |
| 2 | Bor Wildmane (Cenarion Hold) gives **The Calling** quest | One-time chain |
| 3 | Travel to AQ40 entrance + AQ pre-event chain | Silithus → AQ |
| 4 | Receive **AQ40 attune flag** (no item, server-side) | Raid entry unlock |

**Decision-engine rule:** if `Class IN [any] && Cenarion.Reputation >= Friendly && AQ40.Attuned == false`, dispatch to Bor Wildmane. Engine should bundle AQ40 attune with weekly Silithus-Cenarion-rep grind.

## Standing Threshold Map

| Tier | Rep needed (cumulative) | Time estimate (active grind) |
|------|--------------------------|------------------------------|
| Neutral → Friendly | 0 (start) → 3000 | ~5-15 hours (Silithus quests + AQ20 first clear) |
| Friendly → Honored | 3000 → 9000 | ~15-30 hours (Silithus farming + AQ20 weekly) |
| Honored → Revered | 9000 → 21000 | ~30-50 hours (AQ20 + AQ40 weekly + Twilight cultist farming) |
| Revered → Exalted | 21000 → 42000 | ~50-100 hours (sustained multi-week grind) |

**Decision-engine rule:** rep gain rate is **non-linear** — early Friendly→Honored is fast (Silithus quest dump), but Honored→Exalted requires sustained cultist + raid farming.

## Decision-Engine Rules

1. **Friendly priority**: AQ40 attune unlock. Engine should grind Friendly via Silithus quests first.
2. **Twilight Encrypted Text farming**: ~20 rep each, ~5 minutes per pull. Engine should always-loot Encrypted Texts during Silithus grinding.
3. **Twilight Cultist Robe set**: ~75 rep per piece + ~3 pieces per drop chain. Engine should specifically target Cultist mobs.
4. **AQ20 weekly clear**: ~500-1000 rep per clear; 3-day lockout = 2x per week. Engine should auto-queue.
5. **AQ40 weekly clear**: ~3000-5000 rep per clear; 7-day lockout = 1x per week. Engine should align with raid week.
6. **Honored recipe rotation**: at Honored, distribute recipes across alts (1 per profession). Engine should track per-character recipe acquisition.
7. **Druid Pristine Hide turn-in**: 250 rep one-time + class chain reward. Engine should auto-pickup for Druid alts.
8. **Cenarion Vestments planning**: BiS Tailor cloth chest pre-raid; engine should plan multi-prof coordination at Revered.
9. **Reagent farming**: Black Lotus + Mooncloth + Bloodvine cross-recipe planning. Engine should coordinate across multiple alts.
10. **Exalted multi-month grind**: engine should not block on Exalted until late raid weeks; Honored is the practical baseline for raiders.
