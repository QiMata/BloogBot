## Identity

**Upper Blackrock Spire is the most multi-purpose dungeon in vanilla 1.12.1.** It's the **only 10-man dungeon** (between standard 5-mans and 20-man ZG / 40-man raids) and gates **four separate raid attunements** plus drops several pre-raid BiS items.

**Attune chains gating through UBRS**:
- **MC attunement** (Drakkisath kill drops [Drakkisath's Brand quest item](../attunements/molten-core.md))
- **Onyxia attunement** (Drakkisath kill drops Blood of the Black Dragon Champion for both [Alliance](../attunements/onyxia-alliance.md) and [Horde](../attunements/onyxia-horde.md))
- **BWL attunement** (Drakkisath's Brand orb behind boss interaction — see [blackwing-lair.md](../attunements/blackwing-lair.md))
- **Onyxia Horde For The Horde!** quest (Rend Blackhand kill — see [onyxia-horde.md](../attunements/onyxia-horde.md))

**A single UBRS run can complete up to 4 attunement steps simultaneously** if all relevant quests are pre-accepted before pulling Drakkisath.

| Feature | Value |
|---|---|
| Group size | **10-player** (post-patch-1.10) |
| Level required | 58-63 (60 standard) |
| Key/attune required | **Seal of Ascension** for direct entry OR 5+ player ritual at door |
| Lockout | **5-day raid lockout** (treated as raid in 1.12.1 despite 10-man size) |
| Full clear time | 1.5-2.5 hours (well-geared); 3-4 hours (progression) |
| Boss count | 7 named bosses + Solakar Flamewreath summon event + Jed Runewatcher rare spawn |
| Location | Upper section of Blackrock Spire (BRS), accessed via Orb of Command at the front gate |

## Entry mechanics

**Two paths into UBRS**:

1. **Seal of Ascension ring** in inventory: Use the ring at the UBRS door to teleport directly inside. Ring is BoP, soulbound to character. Acquired via the [Seal of Ascension chain](../attunements/seal-of-ascension.md) (Vaelan + 3 LBRS gemstones + Emberstrife Dustwallow).
2. **5-player ritual at door**: 5 players form a circle at the UBRS door + cast in sequence; works without Seal of Ascension. Used by groups when not all members are attuned.

**Engine planning**: Engine prefers Seal of Ascension where possible (faster, no group ritual needed); falls back to 5-player ritual when carrying unattuned alts.

## Run patterns

| Run type | Goal | Time |
|---|---|---|
| **Standard full clear** | All 7 bosses + Solakar event | 1.5-3 hours |
| **Drakkisath rush** | Skip optional events; rush to Drakkisath for attune turn-ins | 45-90 minutes |
| **Multi-attune bundle** | All 4 attunes (MC, Onyxia, BWL, For The Horde) in single run with all quests pre-accepted | 1.5-3 hours (full clear) |
| **T0 chest farm** | Drakkisath kills for RNG T0 chest pieces | Multiple runs (5-day lockout limits) |

**Engine optimization**: Engine plans **multi-attune bundle** runs whenever multiple attune chains are in progress. Failure to pre-accept all quests before Drakkisath = 5-day lockout wasted.

## Decision-Engine Rules

- **id:** `dungeon.ubrs.attune-multi-bundle` — IF `RaidScheduledMC || RaidScheduledOnyxia || RaidScheduledBWL` AND `UBRS run scheduled` THEN engine pre-accepts all 4 relevant quests (Attunement to the Core, Drakefire Amulet, Blackhand's Command, For The Horde Horde-only) BEFORE entering UBRS. Priority **999** (don't waste 5-day lockout).
- **id:** `dungeon.ubrs.seal-of-ascension-prereq` — IF `UBRS run scheduled && !Items.Contains(SealOfAscension) && PartyMembersWithSeal < 5` THEN suspend OR plan 5-player door-ritual setup. Priority **820**.
- **id:** `dungeon.ubrs.t0-chest-farm` — IF `Player.HasNoT0Chest && Class IN {targetClass for chest drop}` AND `!UBRS.LockoutActive` THEN UBRS run priority **600** (gear improvement; 5-day lockout limits frequency).
- **id:** `dungeon.ubrs.briarwood-reed-flag` — IF `Class.IsCaster && JedRunewatcher.IsAlive && !Items.Contains(BriarwoodReed)` THEN engine flags Jed-rare-spawn priority + planned-run. Priority **400** (RNG; deferrable).
- **id:** `dungeon.ubrs.dal-rend-sword-set` — IF `Class IN {Warrior, Rogue Combat Swords} && !Items.Contains(DalRendsSacredCharge AND DalRendsTribalGuardian)` THEN UBRS Drakkisath kill is priority **600** (gear improvement).
- **id:** `dungeon.ubrs.brand-orb-after-drakkisath` — IF `QuestsInProgress.Contains(BlackhandsCommand) && Drakkisath.IsDead && AtBackOrbChamber` THEN click Drakkisath's Brand orb. Priority **820** (combat-time after kill, immediate completion of BWL attune).
