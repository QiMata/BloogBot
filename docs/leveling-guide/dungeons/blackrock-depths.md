## Identity

**Blackrock Depths is the most-trafficked dungeon in vanilla 1.12.1.** It's the **largest 5-man instance in the game** (~4-6 hour full clear), spans **17 named bosses** across multiple wings (Detention Block, Halls of Crafting, Lyceum, Imperial Hall, Black Iron Forge, Grim Guzzler tavern), and is the **central hub for endgame objectives**:

- **MC attunement Core Fragment** (see [molten-core.md](../attunements/molten-core.md))
- **Onyxia attune Marshal Windsor rescue + Argelmach + Angerforge fights** (see [onyxia-alliance.md](../attunements/onyxia-alliance.md))
- **Master's Key + Shadowforge Key** chains (see [5-man-keys.md](../attunements/5-man-keys.md))
- **Hand of Justice trinket** (random chest drop — Warrior/Rogue/Hunter pre-raid BiS)
- **Ironfoe 1H mace** (Hurley Blackbreath drop — Warrior/Rogue/Pal pre-raid BiS)
- **Brutality Blade 1H sword** (Lord Roccor 1% drop)
- **Mirah's Song 1H sword** (Princess Moira drop)
- **Drillborer Disk shield** (Plugger Spazzring drop — tank pre-raid BiS)
- **Spectral Chalice quest** (Mining recipe — kill the Seven Dwarves)
- **Princess kill + Quel'Serrar prep** (Marshal Windsor needs all of this)

A character running BRD multiple times can complete **6-8 separate objectives in parallel**.

## Entrance + access

- **Location**: Inside Blackrock Mountain (BRM), accessed from Burning Steppes (south side via Dark Iron Highway / Door of Heroes) OR from Searing Gorge (north side)
- **Entrance**: Internal portal in BRM lobby; walk down ramps OR use BRM orb (Master's Key required)
- **Level range**: 52-60 (most efficient at 56-60)
- **Key/attune required**: None for entry; **Shadowforge Key** required to bypass parts of the inner area (Grim Guzzler tavern + Imperial Hall full path)

## Recommended group composition

| Role | Class options | Notes |
|---|---|---|
| Tank | Warrior (best), Druid Bear (off-spec), Paladin Protection (Alliance) | Many trash packs are Dark Iron warriors with melee + caster mix; tank needs ~5500 HP unbuffed |
| Healer | Priest, Paladin, Druid, Shaman (Horde) | Long fights; Mana Spring/Mana Tide useful; Resto preferred |
| DPS ×3 | Mage (CC + AoE), Warlock (Curse of Elements + summon), Rogue (lockpick + Pickpocket), Hunter (kite Plugger), Warrior (Fury or Arms DPS), Druid Cat | Mage/Warlock CC required for 4-5 mob pulls; Rogue/Druid stealth-skip useful for Core Fragment runs |

**5-man minimum**; 4-man possible with strong group + Mage CC + Rogue stealth, but slow.

## Run patterns

| Run type | Goal | Time | Path |
|---|---|---|---|
| **Bar Run** | Hand of Justice + Drillborer Disk (Plugger) | 30-45 min | Detention Block → bar (skip Imperial Hall path) |
| **Princess Run** | Princess Moira kill + Mirah's Song | 60-90 min | Detention → Halls of Crafting → Imperial Hall → Princess |
| **Dagran (full clear)** | Final boss + all drops | 3-5 hours | Full clear |
| **Onyxia attune** | Windsor rescue + Argelmach + Angerforge + Jail Break! | 90-120 min | Detention → Imperial Hall (kills) → escort out |
| **MC attune Core Fragment** | Click Core Fragment near MC portal | 30-60 min | Stealth/sneak past trash to MC entrance chamber |
| **Spectral Chalice (Mining)** | Kill the Seven | 60-90 min | Detention → Imperial Hall → Seven chamber |
| **Hurley Run** | Ironfoe drop attempts | 60-90 min | Detention → bar → barrel break → Hurley |

**Engine planning**: A single BRD trip can complete **3-5 quest objectives simultaneously** if planned with all quests pre-accepted. Engine should bundle Onyxia attune + MC attune + Hand of Justice + Princess + Hurley into one or two extended trips per character.

## VMaNGOS / private server notes

- **Plugger Spazzring** is correctly scripted as bartender (neutral until aggroed).
- **Marshal Windsor escort (Jail Break!)** has historically been the most-bugged BRD encounter on private servers; engine plans 1-2 attempts. `[verify VMaNGOS scripting status pass 3]`
- **Spectral Chalice quest spawn** after the Seven Dwarves kill works correctly.
- **Princess Moira rescue/fight branch** is correctly scripted.

## Decision-Engine Rules

- **id:** `dungeon.brd.bundle-objectives` — IF `BRD run scheduled` THEN engine plans a multi-objective run pre-accepting Onyxia attune + MC Core Fragment + relevant farms (Hand of Justice / Brutality Blade / Ironfoe / Plate of the Defender). Priority **820** (logistics optimization).
- **id:** `dungeon.brd.master-key-prereq` — IF `Level>=52 && BRD planned && !KeysInBags.Contains(MastersKey)` THEN consider acquiring Master's Key first (saves BRM walk on subsequent runs). Priority **decision-request**.
- **id:** `dungeon.brd.shadowforge-key-required-for-bar` — IF `BRD run targets Grim Guzzler / Plugger / Hurley && !KeysInBags.Contains(ShadowforgeKey)` THEN require Shadowforge Key first OR bring Rogue with lockpicking. Priority **820**.
- **id:** `dungeon.brd.windsor-rescue-after-key` — IF `Onyxia attune in progress && KeysInBags.Contains(PrisonCellKey)` THEN proceed to Marshal Windsor's cell. Priority **800**.
- **id:** `dungeon.brd.spectral-chalice-mining-class` — IF `Class.Mining >= 230 && !Recipes.Contains(SmeltDarkIronOre)` THEN bundle Spectral Chalice quest with BRD run. Priority **600**.
- **id:** `dungeon.brd.princess-rescue-faction-choice` — IF `Faction-specific Princess quest active` THEN respect rescue branch (Alliance can rescue, Horde fights). Priority **decision-request**.
