# Faction City Services — Vendor + Trainer Matrix Per City Per District

Each of the 6 faction capitals + 4 neutral goblin ports has a consistent service set: **Innkeeper** (hearthstone), **Banker**, **Mailbox**, **Reagent vendor**, **Armor/Weapon Repair**, **Stable Master** (Hunter), **Auctioneer**, **Class trainers**, **Profession trainers**, **Battlemaster** (PvP queue NPC at L60+). Some services are district-specific (Reagent vendor in Mage Quarter; Class trainer hubs in respective race-class districts). **Cross-city service comparison** matters for: faster bracket-1 hearthstone bind, profession progression at Apprentice→Artisan tiers, class quest dispatch routing.

---

## Universal Service Categories

| Service | Purpose |
|---------|---------|
| **Innkeeper** | Hearthstone bind point + lodging atmospheric |
| **Banker** | Bank slot access + bag expansion |
| **Mailbox** | Send/receive mail (1h delivery, cross-faction blocked) |
| **Reagent vendor** | Class-specific reagents (Mage portal reagents, etc.) |
| **Armor/Weapon Repair** | Repair gear durability (cost varies by gear value) |
| **Stable Master** | Hunter pet management (slots 2+3 unlock at L30) |
| **Auctioneer** | AH listing + browsing |
| **Class trainers** | Per-class spell rank training |
| **Profession trainers** | Per-profession skill rank training |
| **Battlemaster** | PvP queue NPC for WSG/AB/AV (post-1.5) |

## Decision-Engine Rules

1. **Universal services**: every capital provides Innkeeper/Banker/Mailbox/Auctioneer/Repair/Stable/Battlemaster. Engine should plan capital trips for any of these basics.
2. **Class trainer routing**: per-class district routing. Engine should encode class trainer hub per faction city.
3. **Master profession trainer access**: limited; **Daryl Stack** (Stormwind Tailoring), **Hjalmar Anvilmar** (Ironforge BS), **Borgosh Corebender** (Org BS Weaponsmith), **Okothos Ironrager** (Org BS Armorsmith), **Tinkmaster Overspark** (Ironforge Gnomish Engineering). Engine should plan trip per Master tier.
4. **Hearthstone bind by bracket**: progressive bind chain Goldshire/Kharanos/Dolanaar → Sentinel Hill/Auberdine → Lakeshire → Refuge Pointe → Light's Hope Chapel.
5. **Cross-faction goblin port use**: for cross-faction trades + cross-continent transit. Engine should plan goblin AH visits.
6. **Bag slot expansion at Banker**: 4 expansion bags = 26.5g cumulative. Engine should plan progressive purchases at L20+ each.
7. **PvP Battlemaster queue**: at L20+ for WSG, L30+ for AB, L51+ for AV. Engine should auto-queue from any capital.
8. **Tauren class restriction reminder**: Thunder Bluff lacks Priest/Mage/Warlock/Rogue/Paladin trainers. Engine should fail-fast for invalid Tauren-class character.

## Snapshot Fields Needed

```text
Snapshot.Position.Zone == "<capital city>"        // city access signal
Snapshot.Position.SubZone                         // district-specific service routing
Snapshot.Faction                                  // determines Alliance/Horde service access
Snapshot.Class                                    // class trainer routing
Snapshot.Race                                     // race-specific trainer hub (Tauren TB, Troll Org Spirits)
Snapshot.Profession.{type}.Skill                  // profession trainer eligibility
Snapshot.Hearthstone.BindLocation                 // Innkeeper bind state
Snapshot.Bank.SlotsUsed / Snapshot.Bank.BagSlotsExpanded  // banker upgrade signal
Snapshot.Mail.Inbox.Count                         // mailbox check signal
Snapshot.Equipment.Durability                     // repair signal
Snapshot.Pet.SlotsAvailable                       // Hunter stable signal
Snapshot.PvPMatch.QueueActive                     // Battlemaster signal
Snapshot.Gold                                     // service cost reserve
Snapshot.Class IN ("Druid", "Hunter", "Shaman", "Warrior")  // Tauren class restriction
```
