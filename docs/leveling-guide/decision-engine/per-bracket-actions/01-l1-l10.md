# Per-Bracket Actions — Levels 1–10

> **Pass 1 skeleton.** Action menu the engine considers while `snapshot.Level` is in [1, 10]. Concrete quest IDs / hub names land in pass 2 when [classes/](../../classes/) and [zones/](../../zones/) are filled.

## Bracket goals

By the end of this bracket the character should have:

- Cleared the racial starting zone (one of: Northshire / Coldridge Valley / Shadowglen / Camp Narache / Valley of Trials / Deathknell / Sunstrider Isle is *not in 1.12.1* — only 6 starts)
- Picked up the **first major class kit** for the class (see class-identity rules below)
- Hearthstone bound to a logical inn (typically the racial capital's city inn or a hub matching planned 10-20 zone)
- Both bag slots filled with whatever 6-slot bags drop / are available
- 1 silver minimum on hand (mostly to learn rank-1 spells from the trainer)
- For Warriors / Paladins / Shamans / Druids / Hunters / Rogues: at least one **weapon-skill** at value ≥ 5 (gates the lvl 10 weapon trainer interactions)

## Suppressions in this bracket

- No mounts (lvl 40 minimum), no BGs (lvl 10+ for WSG), no dungeons except RFC at the *very* end of the bracket for Horde lvl 10s passing through Orgrimmar.
- No PvP rank progression — honor system rewards meaningful at lvl 11+ in WSG and 51+ in AV.
- No profession beyond apprentice rank (75 cap) — yellow/green skill-ups happen during questing, no dedicated grinding.

## Decision-Engine Rules

- IF `snapshot.Level == 1 && !snapshot.QuestsInLog.Any() && snapshot.LastZoneId == <starting-zone-for-race>` THEN accept first racial quest from the starting NPC, priority **800**.
- IF `snapshot.Class == Hunter && snapshot.Level == 10 && !snapshot.HasPet` THEN follow Hunter pet-tame chain, priority **950**, suspends regular questing until pet is tamed.
- IF `snapshot.Class == Warlock && !snapshot.Spells.Contains(SummonImp)` AND `snapshot.Level >= 1` THEN do "Your First Lesson" → "Voidwalker Master" pre-chain (lvl 1 Imp first), priority **950**.
- IF `snapshot.Level >= 6 && snapshot.RestedXP > snapshot.XPToNextLevel * 0.5` AND no class-identity rule eligible THEN prefer **questing** over **mob grinding** (rest XP makes questing strictly better).
