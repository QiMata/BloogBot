# Per-Bracket Actions — Levels 10–20

> **Pass 1 skeleton.** Action menu while `snapshot.Level` is in [10, 20]. Hub names land in pass 2.

## Bracket goals

- Class identity unlocks: Hunter pet (10), Warlock Voidwalker (10), Druid Bear Form (10), Druid Aquatic Form (16), Mage Polymorph (10 trainer / lvl 14 better rank), Shaman Fire Totem (10) and Water Totem (20), Rogue Pick Pocket / Sap (early class-quest steps), Priest Levitate (20)
- Run **Ragefire Chasm** (lvl 13-18, Horde-only natural location, but Alliance can summon) and/or **Wailing Caverns** (15-25) and/or **Deadmines** (Alliance, 17-26) before dinging out of bracket
- Hearthstone moved to whichever 20-30 hub the engine plans to start (Lakeshire / Loch Modan tail, Stonetalon / Ashenvale tail, Hillsbrad lower)
- First **bag upgrades** (10-slot) — common drop during quest rewards in the bracket
- Open Warsong Gulch (lvl 10+) for any character flagged as a PvP toon

## Decision-Engine Rules

- IF `snapshot.Class==Druid && snapshot.Level>=10 && !snapshot.Spells.Contains(BearForm)` THEN run the Bear Form chain (Dolanaar/Bloodhoof for racials → Moonglade), priority **950**. Suspends questing until complete.
- IF `snapshot.Level >= dungeonMinLevel(<dungeon>) - 1` AND a guild/party run is queued AND `!snapshot.IsInCombat` THEN prefer **dungeon entry** over outdoor questing, priority **700**.
- IF `snapshot.Faction == Alliance && snapshot.Level >= 17 && !snapshot.QuestsCompleted.Contains(<deadmines-completion-quest-id>)` THEN park Deadmines on the action menu with priority **680** (drops below mandatory class quests).
- IF `snapshot.HearthstoneBoundZoneId == <starting-zone>` AND `snapshot.Level >= 14` THEN re-bind hearthstone to next-bracket hub on next visit, priority **600**.
