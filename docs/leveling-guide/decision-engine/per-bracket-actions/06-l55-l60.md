# Per-Bracket Actions — Levels 55–60

> **Pass 1 skeleton.** Action menu while `snapshot.Level` is in [55, 60]. The dungeon-spam endgame push and immediate post-60 pre-raid prep.
>
> This bracket has the **highest decision density** in the entire leveling guide — the engine has many parallel objectives (level the last 5, prep attunements, fund epic mount, push profession to 300, build pre-raid BiS, start AD/CC/Hydraxian rep grinds). Tie-breakers in [leveling-priority.md](../leveling-priority.md) earn their keep here.

## Bracket goals (lvl 55-59)

- Final dungeon spam to 60: **LBRS** (55-60), **DM N/W/E** (58-62), **UBRS** (58-63), **Stratholme Live + UD** (58-63), **Scholomance** (58-63), **BRD** (52-60)
- Get **Skeleton Key** (Scholo) and **Shadowforge Key** (BRD) — universal endgame keys
- Push primary professions to **300 (artisan)** if not already
- Top up gold reserves toward 1000g for **lvl 60 epic mount training**

## Bracket goals (lvl 60)

The instant the character dings 60 the engine's planning horizon expands dramatically:

- **Buy epic mount + training (~1000g total, 100% speed)** as soon as gold permits
- Run **MC attunement chain** (UBRS → Drakkisath head → Lothos Riftwaker)
- Run **Onyxia attunement chain** — Alliance: Marshal Windsor (BRD prison rescue, to UBRS for Drakkisath head, back to Stormwind for the masquerade event); Horde: Eitrigg → Rexxar → Onyxia chain (Dustwallow Marsh + UBRS)
- Run **BWL attunement** at the Vaelastrasz orb in BRS upper (requires lvl 60 + UBRS reach)
- Run **AD rep grind** (EPL/WPL Scourgestones, Light's Hope quartermaster) for **Naxx attunement** prep
- Begin **pre-raid BiS** dungeon-set runs (Devout/Wildheart/Beaststalker/Magister/Magecraft/Dreadmist/Defias/Dal'Rend's etc.)
- **Class epic mount chains** for Paladin (Charger) and Warlock (Dreadsteed)
- **Class epic weapon chains**: Hunter Rhok'delar/Lok'delar, Priest Benediction/Anathema, Warrior Quel'Serrar (DM tribute event)

## Suppressions

- No raid attunements before 60 (most are level-gated by their entrances anyway).
- AQ40 / Naxxramas content is gated to 60 + specific attunements; Naxx attune requires AD **Revered**.
- AQ event (Scepter of the Shifting Sands) is server-first; engine should consult a `world.scepter-of-shifting-sands` flag before scheduling AQ40-specific actions on a server where the event has not occurred.

## Decision-Engine Rules

- IF `Level==60 && CopperOnHand >= epicMountCost(faction-discount, class) && !Mounts.Contains(epic60)` THEN buy epic riding + epic mount, priority **850**. **Single highest-impact non-class purchase in the game.**
- IF `Level==60 && !Attunements.Contains(MoltenCore)` AND `KeysInBags.Contains(DrakkisathHeadOrLothosToken)` THEN immediately turn in to Lothos Riftwaker, priority **820**.
- IF `Class==Paladin && Level==60 && CopperOnHand >= chargerCost` THEN start Charger chain, priority **980** (suspends questing — chain involves Stratholme, Scholomance, instanced Paladin lounge in SW).
- IF `Class==Warlock && Level==60 && CopperOnHand >= dreadsteedCost` THEN start Dreadsteed chain, priority **980** (Scholomance summoning ritual + Dire Maul N for the boss summon ingredient).
- IF `Reputation[ArgentDawn] < Revered && Level==60` THEN add EPL/WPL Scourgestone turn-ins to action menu at priority **400-450** depending on travel cost; opportunistic, not exclusive.
- IF `world.scepter-of-shifting-sands == NotStarted && AccountFlag.AQEventInterest` THEN park the Scepter chain (lvl 60 questline beginning in Silithus) on the menu at priority **300** — it's a months-long grind on a real server, not a tick-level decision.
