# Per-Bracket Actions — Levels 30–40

> **Pass 1 skeleton.** Action menu while `snapshot.Level` is in [30, 40].

## Bracket goals

- **Buy apprentice riding skill + first mount at exactly lvl 40** — primary milestone of the bracket. Target 90s for skill + mount cost (varies by faction discount; engine should consult `Reputation[home-city]` for the 10-20% discount tiers).
- Class identity: **Druid Travel Form (30)**, **Hunter Aspect of the Cheetah** (30), **Warlock pet bonuses chain**, **Paladin Lay on Hands rank-ups**, **Mage Polymorph: Pig (30 trainer in 1.12)**, **Priest Mind Vision (30 quest chain)**
- Run **Razorfen Downs** (37-46), continue **Scarlet Monastery** wings (28-45), open **Uldaman** (41-51) at top of bracket
- Push primary professions to **journeyman (150)** then **expert (225)** by end of bracket; Engineering hits the **Goblin / Gnomish split at 200**
- Open **AV** queue is *not yet* available (lvl 51+)

## Decision-Engine Rules

- IF `Level==40 && CopperOnHand >= ridingApprenticeCost(faction-discount) && !Mounts.Any()` THEN travel to riding trainer (faction-specific) and buy training + mount, priority **800**. **Do not skip** — 60% speed transforms travel cost for the rest of leveling.
- IF `Class==Druid && Level==30 && !Spells.Contains(TravelForm)` THEN priority **970**, suspends questing. Travel Form (40% outdoor) is worth ~6 hours of /played in saved travel time before lvl 40.
- IF `PrimaryProfessions.Engineering == 200 && !EngineeringSpecialization.IsSet` THEN park on the menu at priority **720**; do NOT auto-pick — emit a decision request to higher-level account planning (Goblin vs Gnomish is a long-term commitment).
- IF `Level >= 40 && Mounts.Contains(apprenticeMount) && !ZonePlanningSwitched(50-60)` THEN start nudging hearthstone / quest plan toward Tanaris/Feralas/STV-upper hubs.
