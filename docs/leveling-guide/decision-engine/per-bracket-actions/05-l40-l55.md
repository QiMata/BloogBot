# Per-Bracket Actions — Levels 40–55

> **Pass 1 skeleton.** Action menu while `snapshot.Level` is in [40, 55]. This is the long mid-game stretch — the bracket where the player is mounted, professions are mid/late, and the path to 60 is mostly questing in the contested 40-55 zones with periodic dungeon insertions.

## Bracket goals

- Continue questing in **Tanaris**, **Feralas**, **STV upper**, **Searing Gorge**, **Badlands**, **Hinterlands**, **Azshara**, **Hillsbrad upper**, **Stranglethorn upper**, **Felwood (lower)**
- Run **Zul'Farrak** (44-54), **Maraudon** (45-55, 3 wings: Purple/Orange/Princess), and start **Sunken Temple** at top of bracket (50-60)
- Push primary professions to **artisan (300)** by ~lvl 50 if mat-rich
- **Open AV (51+)** for any PvP toon
- Begin dungeon-key runs: **Crescent Key** (Sunken Temple), **Master's Key** (BRM)
- Top of bracket: pick up **Argent Dawn rep grind hooks** in EPL/WPL preview quests

## Decision-Engine Rules

- IF `Level >= 51 && PvPRole && !BattlegroundsCompleted[AlteracValley].Any()` THEN open AV queue concurrent with questing, priority **600**. (Honor at 51-59 is wasted on rank progression but earns BG rep + faction items.)
- IF `Level >= 50 && !KeysInBags.Contains(CrescentKey)` AND ST run is on the action menu THEN do **Atal'ai Exorcism** chain first, priority **720**.
- IF `Engineering == 225 && Specialization == Goblin` THEN flag the toon as the engine's "AoE leveler" candidate (Goblin Sapper/Bombs are leveling-relevant in EPL/Felwood).
- IF `Level >= 51 && Reputation[ArgentDawn] < Friendly` THEN take low-cost AD intro quests during EPL/WPL passes (priority **400** — does not displace primary quest hubs).
