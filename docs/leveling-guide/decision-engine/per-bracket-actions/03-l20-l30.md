# Per-Bracket Actions — Levels 20–30

> **Pass 1 skeleton.** Action menu while `snapshot.Level` is in [20, 30].

## Bracket goals

- **First mount at 40** is on the horizon — start banking gold (target 90s for apprentice riding training + ~80s for the mount itself; flexible by faction discount). Earlier is better; questing this bracket should fund it.
- Class identity: Druid Cat Form (20), Mage Conjure Water rank-up, Warlock Felsteed (40 actually — leave for next bracket), Shaman Ghost Wolf (20), Priest Levitate (20)
- Run **Shadowfang Keep** (22-30), **Stockades** (24-32 Alliance only), **Blackfathom Deeps** (24-32), **Gnomeregan** (29-38 — top of bracket / spillover), **Razorfen Kraul** (30-40 — spillover)
- Open **Arathi Basin** (lvl 20+) for PvP toons
- Pick up second profession if not already done; both at apprentice (75) cap and pushed to journeyman (150)

## Decision-Engine Rules

- IF `Class==Druid && Level>=20 && !Spells.Contains(CatForm)` THEN run Cat Form chain (multi-zone quest), priority **940**. Costs 1 hour of /played but unlocks 30%-faster outdoor traversal until Travel Form (lvl 30).
- IF `Class==Druid && Level>=30 && !Spells.Contains(TravelForm)` THEN swap to Travel Form quest from trainer, priority **950** at the moment of dinging 30. (Anchored here for visibility; covered in next bracket.)
- IF `goldOnHand >= mountFundTarget(40)` AND `Level >= 38` THEN suppress non-essential profession spending; bank all loot for mount purchase.
- IF `PvPRole && Level >= 20 && !BattlegroundsCompleted[ArathiBasin].Any()` THEN open AB queue concurrent with questing, priority **600**.
