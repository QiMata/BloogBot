# L50-L60 Bracket — Endgame Outdoor + 5-Man Spam + L60 Class Epics

The final bracket and the most multi-objective. From L55 onward the character is **functionally lvl 60** for purposes of dungeon access (5-man instances scale down well). The bracket goal pivots from "level up" to "**ding 60 already attuned + geared + repped**". This is the longest bracket by /played time (~25-50 hours typical), driven by:
- 8 zones with active questing (Un'Goro, Felwood, WPL, EPL, Burning Steppes, Winterspring, Silithus, Blasted Lands)
- 9 dungeons in the 55-60 spam window (BRD, LBRS, UBRS, Strat-Live, Strat-UD, Scholo, DM-N/W/E)
- 4-7 reputation grinds (Argent Dawn, Cenarion Circle, Hydraxians, Thorium Brotherhood, Timbermaw, Wintersaber, Steamwheedle)
- 5 raid attunements (UBRS Seal of Ascension, MC Core Fragment, Onyxia A/H, BWL Drakkisath's Brand, Naxx AD-Honored)
- 7 class epic chains (Pal Charger, Warlock Dreadsteed, Hunter Rhok'delar, Priest Benediction, Warrior Quel'Serrar, Druid Pristine Hide T0.5, Mage Robe of the Archmage tailoring)

This section is the gateway between leveling and raiding. Pre-raid BiS planning happens here. PvP rank progression starts here.

---

## Bracket Goals (target end-state at L60)

- [ ] All 51 talent points spent (10 across L51-L60; final-level talent at 60 is the 51st point)
- [ ] At least 1 class epic chain completed if applicable (Pal/Warlock mount; Hunter/Priest weapon)
- [ ] UBRS attunement (Seal of Ascension) acquired — gates 4 raid attunements
- [ ] Onyxia attunement chain completed (Drakefire Amulet)
- [ ] MC attunement (Core Fragment in BRD) — same UBRS run can bundle this
- [ ] BWL attunement chain start (Vaelastrasz orb requires Drakkisath kill in UBRS already)
- [ ] AD reputation Honored (Naxx attune cost reduction or zero)
- [ ] At least 5+ dungeons cleared from the 55-60 spam window
- [ ] Profession primary skill 300 (master) on at least 1 of 2 primaries
- [ ] First-aid 300 (Heavy Runecloth bandage)
- [ ] Hearthstone bound to **Light's Hope Chapel** (EPL) for AD turn-ins, or Cenarion Hold (Silithus) for AQ/Cenarion grinds
- [ ] Gold reserve 200-500g (epic mount fund: 1100g target)

## Pre-Raid Attunement Chain Map

```
Vaelan Seal of Ascension (Burning Steppes, Lvl 57+)
    │   chains: gather 3 LBRS gemstones + Emberstrife subjugation in Dustwallow
    ↓
UBRS access (Drakkisath kill)
    │   bundles 4 attunes:
    ├──→ MC attune (Drakkisath head turn-in to Lothos Riftwaker in BRD lava lake)
    ├──→ Onyxia A: Drakkisath head + Stormwind chain (Marshal Windsor)
    ├──→ Onyxia H: Drakkisath head + Eitrigg/Rexxar chain (Thrall + Rexxar Desolace)
    └──→ BWL: Drakkisath's Brand orb (touched in UBRS) → Vaelastrasz orb in BRS upper

AD Honored at Light's Hope Chapel
    │   accumulate Scourgestones + Plague-related quests (15-25 hours)
    ↓
Naxx attune (Naxx40 access at lvl 60 — Honored cost 60g; Revered 30g; Exalted free)

Cenarion Circle Friendly at Cenarion Hold
    │   complete ~10 quests in Silithus
    ↓
AQ20 / AQ40 attune (no rep gate for entry; AQ40 attune chain unlocks at Friendly)
```

**Decision-engine rule:** Vaelan chain is the gateway. Engine should never enter MC/Ony/BWL questing without UBRS Drakkisath head ready. AD Honored is parallelizable with UBRS spam since both are L55-60 outdoor + dungeon grinds.

## Bracket Exit Conditions (= L60 Raid-Ready)

When all of the following are true, engine should mark character "Raid-Ready":

- [ ] `Level == 60`
- [ ] `Talents.PointsSpent == 51` (full 51-point allocation)
- [ ] `Mounted == "Epic"` (Journeyman Riding + 100% mount, OR class epic mount free)
- [ ] `Attunements.UBRS == true` (Seal of Ascension)
- [ ] `Attunements.MC == true` (Core Fragment turn-in)
- [ ] `Attunements.Onyxia == true` (Drakefire Amulet)
- [ ] `Attunements.BWL == true` (Vaelastrasz orb)
- [ ] `Reputation.ArgentDawn >= Honored` (Naxx attune cost reduction)
- [ ] `Reputation.CenarionCircle >= Friendly` (AQ40 access)
- [ ] `Profession.Primary.Skill == 300` (at least 1 of 2 primaries at master)
- [ ] `Dungeons.Cleared.CountOfL55toL60 >= 5` (BRD/UBRS/Strat-Live/Strat-UD/Scholo/DM-N/W/E — any 5)
- [ ] `Equipment.PreRaidBiS.Slots >= 14 / 17` (T0 + dungeon blues + Tailoring/LW crafted)
- [ ] `Gold >= 50g` (consumable reserve for raid week)

If `Level == 60 && WallClock.SinceL60Ding > 50h && Equipment.PreRaidBiS.Slots < 10`, engine flags **gear lag** — character is L60 but unprepared for raids. Likely cause: skipped class epic quest, missed AD rep, or missed dungeon spam window.

## Decision-Engine Rules

1. **At L50 ding**, dispatch class epic chain pre-quest priority (Hunter Demon Stalker pre, Warlock Dreadsteed pre at Strahad Farsan).
2. **At L55**, dispatch UBRS Seal of Ascension chain (Vaelan in Burning Steppes) before any other endgame priority.
3. **At L58**, queue UBRS group with Drakkisath head + 4 attunes bundling.
4. **AD Honored gate**: engine should plan ~10-30 hours of Scourgestone + Plagueland quests at L55+ before MC entry (Naxx attune is a downstream goal, but AD Honored unlocks Trinkets used in MC).
5. **Pristine Hide → T0.5 (Druid)**: engine should always pickup the chain at L50 even if T0.5 deferred to L60. Pristine Hide drops are RNG and bracket-defining.
6. **Class epic OVERRIDE**: Pal Charger / Warlock Dreadsteed / Hunter Rhok'delar / Priest Benediction / Warrior Quel'Serrar must complete before MC/Ony/BWL raid entry. Engine should fail-fast if `Class IN [Pal,Wlk,Hun,Pri,War] && Level == 60 && ClassEpicComplete == false && AttemptingRaidEntry == true`.
7. **Silithyst world PvP (1.12)**: opportunistic — engine reads `Snapshot.NearbyResources.Has("SilithystMound")` and toggles Silithyst-bearer mode if not in combat or in a group of 1.
8. **Cenarion Circle Friendly gate**: engine should grind ~10 Silithus quests at L57-58 before AQ40 attune attempt.
9. **Lord Kazzak prep**: at L60, Kazzak fight requires 30+ raid group. Engine should never solo-queue Blasted Lands at L60 questing without first checking `RaidGroup.Active == true`.
10. **Profession 300 push**: at L60, profession 300 is a bracket-exit gate. Engine should pause raid attune farming for 4-8 hour profession-finish blocks if `Skill < 285`.
11. **Hearthstone-Light's-Hope mandatory**: at L55+, engine should rebind to Light's Hope Chapel for AD turn-in efficiency. Cost = 1 hearth CD (~1 hour); benefit = entire bracket-end gameplay.
12. **Stratholme Baron 45-min run**: timed event — engine should attempt only if group is geared (avg item lvl 55+). Reward: Eye of Naxxramas trinket + Deathcharger's Reins (1% drop) + T0 set continuation.
