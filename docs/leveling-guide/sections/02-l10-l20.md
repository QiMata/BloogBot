# L10-L20 Bracket — Quest Hubs, Class Unlocks, First Dungeon

The "graduation" bracket. The character leaves the racial starter zone, picks up the first talent point, completes the **mandatory class identity quest** (pet/voidwalker/Bear Form/totems/Verigan's Fist), and is gated for the **first instanced dungeon** (Deadmines / Ragefire Chasm / Wailing Caverns / Shadowfang Keep / Blackfathom Deeps). By L20 the character is ready for the L20-L30 bracket and the first major systems milestone — buying first regular mount training is still 20 levels away (lvl 40), but the "second rank of class spells" thinning of the spellbook starts here.

---

## Bracket Goals (target end-state at L20)

- [ ] First talent point spent at L10, ten more spent across L11-L20
- [ ] Class identity quest completed (pet for Hunter, Voidwalker for Warlock, Bear for Druid, totems progression for Shaman, Verigan's Fist for Paladin)
- [ ] At least 1 instanced 5-man cleared (Deadmines / RFC / WC / SFK)
- [ ] Hearthstone bound to the **best central inn** for the next bracket (Sentinel Hill→Lakeshire / Crossroads→Tarren Mill)
- [ ] All bracket-appropriate flight paths attuned
- [ ] First profession recipes/patterns at skill ~75-100; second profession picked up if not done in L1-L10
- [ ] First-aid bandage rank advanced to Heavy Linen / Wool (skill ~75)
- [ ] All starter-zone-tier gray gear vendored, replaced with green-tier quest rewards / dungeon drops

## Bracket Milestones — Per Level

| Level | Mandatory action | Why it matters |
|-------|------------------|----------------|
| 10    | First talent point at trainer; **start class identity quest** | Talent tree opens entire build path; class quests are mandatory not optional |
| 10    | Hunter: `Tame Beast` quest chain at race trainer | Without pet, Hunter DPS drops ~40% |
| 10    | Warlock: `Summon Voidwalker` chain at city trainer | Tank pet; soloing without it requires Drain-Tanking with Imp |
| 10    | Druid: `Moonglade` quest from race trainer → Bear Form | Bear Form ~3.5x effective HP for soloing tough mobs |
| 10    | Shaman: `Call of Fire` chain at trainer → Fire Totem | Fire Nova totem is bracket-defining mob-clear |
| 10    | Paladin: pickup `The Tome of Divinity` chain | Sets up L12 Verigan's Fist 2H mace |
| 10    | Priest: race-specific spell trainer for *Devouring Plague / Hex of Weakness / Starshards / Feedback / Elune's Grace* | Power spike for solo questing |
| 12    | Paladin: turn in `The Test of Righteousness` → **Verigan's Fist** 2H mace | Best-in-bracket weapon, used through L25+ |
| 14    | Druid: `Cure Poison` quest *(skip)* | Strictly worse than train-bought Abolish Poison at L26 |
| 16    | Druid: `Aquatic Form` chain (Moonglade) | Required for low-level water zones (Wetlands, Swamp of Sorrows) |
| 16    | Hunter: pet skill points fill out (Bite/Claw/Growl/Dash) | Pet builds are bracket-defining; respec-pet via beast-trainer is free |
| 18-20 | First mount **savings** target (~6g for L40 mount + 80g training) | Start hoarding gold; target ~100g by L40 |
| 20    | Shaman: `Vision Quest` chain → Water Totem | Mana Spring tier-1; healing windfall for group play |
| 20    | Warlock: `Devourer of Souls` chain → Succubus | Succubus is solo DPS pet; Voidwalker is tank pet |
| 20    | Paladin: `Redemption` chain → res ability | Combat utility; required for group play |
| 20    | Priest: rank-2 racial-specific spell available | E.g., rank-2 Devouring Plague (Undead) at L20 |
| 20    | Mage: `Polymorph` rank-1 still serviceable; Frost Bolt rank-3 trained | Frost build power spike for AoE prep |

## Alliance Zones (10-20)

### Westfall (lvl 9-15)

| Field | Value |
|-------|-------|
| Continent | Eastern Kingdoms |
| Capital hub | **Sentinel Hill** (innkeeper, FP to Stormwind/Lakeshire/Darkshire) |
| Sub-hubs | Saldean's Farm (Furlbrow's Pumpkin Patch FP), Moonbrook (Defias-controlled, no inn) |
| Dungeon | **Deadmines** (lvl 17-20) — entrance in Moonbrook |
| Faction enemies | Defias Brotherhood, Riverpaw gnolls, Harvest Watchers, Old Murk-Eye (rare) |
| Race fit | Human / Dwarf / Gnome / Night Elf — all lvl 10 ports converge here |
| Quest chains | Westfall Brigade chain (Marshal Stoutmantle), Defias Tower scout, Goretusk Liver Pie |

**Key NPCs:**
- Marshal Gryan Stoutmantle — main quest hub at Sentinel Hill watchtower
- Captain Danuvin — guard captain, gives Riverpaw quests
- Farmer Saldean — Saldean's Farm hub, gnoll quests
- Salma Saldean — chicken / dog food chain (low XP, easy)

**Decision-engine cue:** if `Faction == Alliance && Level ∈ [10, 15] && PartyMembers.Count == 0`, Westfall has the highest quest density of any L10-15 Alliance zone. Loch Modan is alternate for dwarves/gnomes.

**Decision-engine cue:** Crossroads is the highest quest-density hub in the game for L10-L20 Horde. WC dungeon attunement chain (`Disciple of Naralex` reward) is a free green; bots should always run it before zoning out.

## L10-L20 Dungeon Run Order

| Order | Dungeon | Faction | Min lvl | Optimal lvl | Notes |
|-------|---------|---------|---------|-------------|-------|
| 1 | **Ragefire Chasm** (Org) | Horde | 8 | 14-18 | 4 bosses, 30-min runs; Tribal Belt cloth recipe drop |
| 1 | **Wailing Caverns** | Horde primary | 15 | 18-22 | 8+ bosses, very long (90-min); naturalist's quest set |
| 2 | **Deadmines** | Alliance | 15 | 17-20 | 6 bosses; **Cookie's Stirring Rod** healer wand BIS |
| 2 | **Shadowfang Keep** | Both | 18 | 22-26 | 6 bosses; cross-faction; Meteor Shard / Necrology Robes |
| 3 | **Blackfathom Deeps** | Both | 20 | 24-28 | 7 bosses; **Lorgalis Manuscript** mage Frost AoE wand |
| 3 | **Stockades** (SW) | Alliance | 22 | 24-28 | 4 bosses; very short, ~20 min full clear |

**Decision-engine rule:** for the L10-L20 bracket, prioritize **WC (Horde) / Deadmines (Alliance)** as the only mandatory dungeon. Other dungeons can shift to the L20-30 bracket with no XP loss.

## Hearthstone Bind Strategy by Bracket

| Bracket end | Alliance bind | Horde bind |
|-------------|---------------|------------|
| L10-L15 | Sentinel Hill (Westfall) or Auberdine (Darkshore) | Crossroads (Barrens) or Sepulcher (Silverpine) |
| L15-L20 | Lakeshire (Redridge) or Menethil (Wetlands) | Crossroads or Tarren Mill (Hillsbrad) |
| L20 graduation | Stormwind (rebind for Stockades / Duskwood prep) | Orgrimmar (rebind for ZF prep / Camp Mojache) |

**Decision-engine rule:** rebind hearthstone whenever the **next-bracket primary hub** is more than 1 zone away from the current hub. Don't rebind for marginal gains (<5 minutes saved per hearth).

## Class Identity Quest — Per Class L10-L20

| Class | L10 quest | L20 quest | First-talent recommendation |
|-------|-----------|-----------|-----------------------------|
| Warrior | None mandatory | None mandatory | Cruelty (5/5) → Improved Charge (2/2) for Fury, or Improved Heroic Strike (3/3) → Anger Mgmt (Arms) |
| Paladin (A) | The Tome of Divinity (sets up L12) | Redemption + Sense Undead | Improved Blessing of Might (5/5) Ret, or Imp Seal of Righteousness (Holy/Ret) |
| Hunter | **Taming the Beast** chain → Tame Beast spell | Mostly stat-stick gear | Improved Aspect of the Monkey (3/3) Survival → BM tree |
| Rogue | None mandatory; daggers from quest rewards | Lockpicking trainer at L24 | Improved Sinister Strike (5/5) Combat |
| Priest | Race-specific spell @ L10 (e.g., Devouring Plague) | Race-specific spell rank 2 @ L20 | Wand Specialization (5/5) → Improved Power Word: Shield (Disc) for solo |
| Shaman (H) | **Call of Fire** → Fire Totem | **Vision Quest** → Water Totem | Improved Earth Shock (3/3) or Convection (5/5) Ele |
| Mage | None mandatory; Polymorph @ L20 trainer | None mandatory | Improved Frostbolt (5/5) Frost |
| Warlock | **Summon Voidwalker** chain | **Devourer of Souls** → Succubus | Improved Corruption (5/5) Affliction |
| Druid | **Bear Form** chain (Moonglade) | None mandatory; Cat Form @ L20 trainer (no quest) | Improved Mark of the Wild (5/5) Resto-stub or Furor (5/5) Feral |

**Decision-engine rule:** if `Class ∈ {Hunter, Warlock, Druid, Shaman, Paladin} && Level >= 10 && ClassQuestComplete == false`, the engine MUST prioritize the class quest over any zone quest hub. The class quests grant non-substitutable abilities.

## Travel Speed Considerations

- **No mount until L40** (1.12.1 patch — final mount level was reduced from 60 to 40 in patch 1.12.0; engine confirms via `Snapshot.Mounted == false` for entire L1-L40)
- **Druid Travel Form @ L30** (cat-speed 40% on land, free, no quest required in 1.12)
- **Shaman Ghost Wolf @ L20** (Ghost Wolf trained, 40% speed, indoors-OK)
- **Hearthstone 1-hour CD** is the only fast-travel for non-druid/shaman pre-L40
- **Boats/zeppelins** — Menethil↔Auberdine (5 min); Org↔UC (4 min); Org↔Grom'gol (3 min, only after L35 quest chain)

**Decision-engine cue:** `if Class ∈ {Shaman, Druid} && Level >= 20`, prefer overland questing routes over hearthstone-CD planning. For all other classes pre-L40, plan zone-rotation to align with hearthstone CD.

## Bracket Exit Conditions

When all of the following are true, engine should mark bracket complete and transition to L20-L30:

- [ ] `Level >= 20`
- [ ] `Talents.PointsSpent == 11` (1 at L10 + 10 across L11-L20)
- [ ] `ClassQuests.L10Complete && (ClassQuests.L20Complete || Class NOT IN [Pal, Sha, Wlk])`
- [ ] `Dungeons.Cleared.Contains("Deadmines" or "WailingCaverns" or "ShadowfangKeep")` — at least one
- [ ] `Hearthstone.Bind ∈ {bracket-2-recommended-hubs}`
- [ ] `Gold >= 1g` (basic spell-rank training reserve)

If `Snapshot.LevelDelta(start, end) < 10 && Snapshot.WallClock > 8h`, engine flags **bracket stall** — likely cause: missing class quest unlock, profession skill gap, or PvP corpse-camp on a contested-zone road.

## Decision-Engine Rules

1. **At L10 turn-in**, dispatch character to race-trainer; do not advance map until class quest accepted.
2. **For Hunter/Warlock**, do NOT enter Deadmines/WC/SFK without pet — engine should fail-fast with `MissingPetForDungeon`.
3. **For Druid**, prioritize Bear Form quest (Moonglade port → Lunaclaw kill) over the next zone quest hub at L10.
4. **For Shaman**, Fire Totem at L10 and Water Totem at L20 are not optional — gate Wailing Caverns / Razorfen Kraul on Fire Totem completion.
5. **For Paladin**, Verigan's Fist 2H mace (L12 turn-in) is BiS until L25+; do not skip even if grouped with healer.
6. **Hub progression** within a zone is *quest-NPC-driven*, not level-driven — engine reads `Snapshot.QuestLog.Count(zone)` and rotates to next hub when `Available < 2`.
7. **Cross-zone walks** through `level > Snapshot.Level + 5` mob density are forbidden (engine pathfinder should reroute via boat/zeppelin/portal).
8. **PvP-server rule**: Tarren Mill / Southshore road is forbidden for L18-L25 unless `PvP.GroupSize >= 3 && PvP.AvgLevel >= 25`.
9. **Dungeon cooldown**: 1-hour boss reset in 1.12; the 5-instances-per-hour limit is a Classic 2019 change, **NOT** in vanilla 1.12 `[verify pass 3]`.
10. **Gold sink at L20**: spell-rank training costs scale ~10s per rank-2 spell. Engine should enforce `Gold >= 1g` reserve before training rotation.

## Snapshot Fields Needed

```text
Snapshot.Level                       // gates bracket entry/exit
Snapshot.Class                       // determines class-quest path
Snapshot.Race                        // determines starter zone follow-on + race-specific Priest/Hunter
Snapshot.Faction                     // determines Westfall vs Barrens path
Snapshot.Talents.PointsSpent         // bracket exit gate
Snapshot.QuestLog.Active             // hub-progression decision
Snapshot.QuestLog.Completed.Contains("ClassQuest:L10:<class>")    // class identity gate
Snapshot.QuestLog.Completed.Contains("ClassQuest:L20:<class>")    // ditto for L20
Snapshot.Pet.Active                  // Hunter/Warlock dungeon gate
Snapshot.Spells.Has("BearForm")      // Druid bracket signal
Snapshot.Spells.Has("FireTotem")     // Shaman bracket signal
Snapshot.Inventory.Contains("Verigan's Fist")  // Paladin bracket signal
Snapshot.Profession.{type}.Skill     // skill-cap gate per profession
Snapshot.Dungeons.Cleared            // L10-L20 completion gate
Snapshot.Hearthstone.BindLocation    // bracket transition signal
Snapshot.Gold                        // spell-rank training reserve
Snapshot.PvPEnabled                  // contested-zone path-planner
Snapshot.WallClock.SinceLevelStart   // bracket-stall detection
Snapshot.Position.{Zone, X, Y}       // path-planner state
```
