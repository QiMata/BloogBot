# L40-L50 Bracket — Mid-Game Spread, Sunken Temple, Pre-60 Profession Specialty

The "wide bracket". With L40 mount in hand, the character can fluidly travel between **5+ simultaneous quest zones** spanning two continents. Profession specialty unlocks (Blacksmithing Armorsmith/Weaponsmith at 200 + L40, Leatherworking Dragon/Tribal/Elemental at 225 + L40, Engineering Goblin/Gnomish at 200 + L30) become available. The first **multi-wing instances** (Zul'Farrak, Maraudon, Sunken Temple) require coordinated 5-mans rather than carry runs.

---

## Bracket Goals (target end-state at L50)

- [ ] All L40-L50 talent points spent (10 across L41-L50; cumulative 41)
- [ ] Profession specialty selected and pursued (Armorsmith/Weaponsmith + LW spec + Engineering spec)
- [ ] At least 4 instanced 5-mans cleared this bracket (ZF + Maraudon-3-wings + Sunken Temple)
- [ ] **Sunken Temple Crescent Key** acquired (Pusillin chase in DM East entrance area)
- [ ] Hearthstone bound to bracket-end hub (Gadgetzan or Camp Mojache or Aerie Peak/Revantusk)
- [ ] Profession primary skill ~250-275 (Mining → Truesilver, Herbalism → Sungrass, Skinning → Rugged Leather)
- [ ] First-aid bandage rank advanced to Runecloth (skill ~260)
- [ ] Gold reserve 100-200g (start saving for L60 epic mount: 1000g + 100g)
- [ ] All bracket-appropriate flight paths attuned (Hinterlands, Searing Gorge, Feralas, Tanaris all mandatory)

| Level | Mandatory action | Why it matters |
|-------|------------------|----------------|
| 40 | Mount (covered iter 37) — riding skill 75, racial mount | Bracket-defining 60% movement speed |

| 42 | Hunter's Mark rank 5 (auto-trained) | DPS uptick |
| 44 | Hammer of Wrath trained (Paladin) | Sub-20% HP execute, key PvP/raid tool |
| 44 | Aspect of the Pack trained (Hunter) | Group 30%-speed buff, used pre-mount group QoL |
| 46 | Higher-rank caster spells trained (Frostbolt 7, Holy Light 6, etc.) | DPS/healing parity |
| 50 | **Druid**: `Pristine Hide of the Beast` quest at trainer | Reagent-collecting class quest; intro to Cenarion Circle rep grind |
| 50 | **Mage**: Sunken Temple chain — `The Stave of Equinex` reward (Celestial Stave) | Caster staff with major spirit/spelldmg stats |
| 50 | **Priest**: Sunken Temple chain — `Eye of the Dead` trinket | Spell crit + party group-heal proc trinket |
| 50 | **Warlock**: Demonic Sacrifice talent capstone unlock potential | Demonology 31-pt path |
| 50 | **Rogue**: Lockpicking should be 250+ — required for Mara/ST chests | Mara Black Vault chest, ST chest, AH alt-mailing |

## Both-Faction Zones (40-50)

### Tanaris (lvl 40-50)

See [04-l30-l40.md](04-l30-l40.md#tanaris-lvl-40-50--top-of-bracket-transition) for hub overview. Bracket extension: ZF dungeon attune chain via **Hydromancer Velratha** (in ZF herself); pre-attune from Marin Noggenfogger Goblin chain.

**Decision-engine cue:** Tanaris is the primary L40-L48 questing zone for both factions. Gadgetzan mount-test-drive turns into Steamwheedle Cartel rep grind via Booty Bay/Ratchet/Everlook turn-ins.

### Feralas (lvl 40-50)

| Field | Value |
|-------|-------|
| Continent | Kalimdor |
| Hubs (A) | **Feathermoon Stronghold** (innkeeper, FP to Auberdine/Theramore/Nijel's Point; on Sardor Isle, requires hippogryph from Thalanaar) |
| Hubs (H) | **Camp Mojache** (innkeeper, FP to Crossroads/Sun Rock/Freewind/Thunder Bluff) |
| Sub-hubs (neutral) | Thalanaar (FP, Alliance approach), Forgotten Coast (naga), Dire Maul ruins (entrance to L55+ instance) |
| Dungeon | **Dire Maul** (lvl 55-60) — top-of-bracket-transition entry only |
| Faction enemies | Gordunni ogres (Dire Maul), Wildkin/Owlkin, Hippogryph (protected), Sprite Darters (protected by Kindal Moonweaver), Grimtotem tauren (Sprite Darter captors), Lord Shalzaru naga |
| Race fit | Both factions; hippogryph travel from Thalanaar is bracket signature |
| Quest chains | Shandris Feathermoon (A) Sentinel chain; Witch Doctor Uzer'i (H) furbolg chain; Kindal Moonweaver (Sprite Darter rescue, gives Sprite Darter Hatchling pet); Falfindel Waywarder (Thalanaar) Stranglethorn-portal hub |

**Key NPCs (Alliance):**
- Shandris Feathermoon — Sentinel quest hub
- Latronicus Moonspear — naval defense chain (Forgotten Coast)
- Brohann Caskbelly — Hippogryph protection chain

**Key NPCs (Horde):**
- Witch Doctor Uzer'i — Camp Mojache main hub
- Hadoken Swiftstrider — Cenarion Circle introduction (defers to AQ-prep at 60)

**Decision-engine cue:** Feralas is the highest-density L40-50 zone for Horde. Alliance equivalent is Tanaris+Feralas split. Hippogryph from Thalanaar is mandatory before Sardor Isle FP path.

### Searing Gorge (lvl 43-50)

| Field | Value |
|-------|-------|
| Continent | Eastern Kingdoms |
| Capital hub | **Thorium Point** (neutral; innkeeper, FP to Aerie Peak/Stormwind/Booty Bay; **Thorium Brotherhood** rep grind starts here) |
| Sub-hubs | The Cauldron (Dark Iron-controlled excavation), Pyrox Blaze, The Slag Pit |
| Dungeon | None native; **Blackrock Depths** (lvl 52-60) accessible via Blackrock Mountain bridge |
| Faction enemies | Dark Iron dwarves (Cauldron + Blackrock Mountain approach), incendosaurs, golems, fire/lava elementals, Bael'Gar (lvl 50 elite fire elemental) |
| Race fit | Both factions; Thorium Point is fully neutral |
| Quest chains | Master Smith Burninate fiery flux chain (mining-focused), Hansel Heavyhands hostile-fauna culling, Velarok Windblade Dark Iron sentry tower demolition, **Cauldron quests** (Dark Iron operatives by tower — long chain ending in Slag Pit elite kills) |

**Key NPCs:**
- Master Smith Burninate — flux/blacksmithing chain
- Hansel Heavyhands — beast/elemental quests
- Velarok Windblade — Dark Iron espionage chain
- Lothos Riftwaker — Sunken Temple summoner (separate from MC summoner of same role)

**Decision-engine cue:** Searing Gorge is the **Thorium Brotherhood rep gateway**. Engine should always pickup all 6 Thorium Point quests on first visit; Thorium Brotherhood Honored unlocks Sulfuron Hammer recipe (Tier 1.5 quest at 60).

|-------|-------|

### Azshara (lvl 45-55)

| Field | Value |
|-------|-------|
| Continent | Kalimdor |
| Hubs (H) | **Valormok** (small Horde encampment, FP only — no innkeeper) |
| Hubs (A) | **Talrendis Point** (small Alliance outpost, FP only) |
| Sub-hubs | Bay of Storms (south), Forlorn Ridge, Eldarath Ruins (lvl 50+ elite naga), Hetaera's Clutch (hydra elite) |

| Race fit | Both factions sparsely; quest hub density is the lowest of any L45+ zone |
| Quest chains | Loramus Thalipedes demon-hunter chain (multi-zone — links to Felwood/Outland deferral), Spitelash naga suppression, **Azuregos pre-quest chain** (lvl 60 world boss prep, deferred to L60), Highborne artifact gathering |

**Decision-engine cue:** Azshara is **quest-thin**. Best used for Mining (Mithril/Truesilver veins) + Herbalism (Khadgar's Whisker, Firebloom on Bay of Storms shore) gathering rather than questing. Skip if Tanaris+Feralas+Searing Gorge have remaining quests.

| Order | Dungeon | Faction | Min lvl | Optimal lvl | Notes |

| 2 | **Maraudon: Princess wing** (orange/Earth Song) | Both | 40 | 44-50 | Continued from iter 36; full clear includes Vault → Foulspore Cavern (Noxxion + Razorlash) → Princess Theradras + Celebras → Earth Song Falls (Tinkerer Gizlock + Rotgrip + Princess) |
| 3 | **Maraudon: outer (purple)** | Both | 42 | 46-52 | Veng + Earthcaller Halmgar + Tinkerer Gizlock; **Bonecrusher** mace; reagent-grade armor |

## Sunken Temple Attune (Crescent Key)

The Crescent Key is required to open the Sunken Temple's outer gate. Acquisition path:

| Step | Location | Action |
|------|----------|--------|
| 1 | Dire Maul East entrance area | Find **Pusillin** (lvl 44 satyr, runs in a fixed loop) |
| 2 | Pusillin loop | Catch Pusillin (he stops 5 times, hand him items he asks for OR engage and wait until 30%) |
| 3 | Final stop | Pusillin gives `Pusillin's Letter` and `Crescent Key` |
| 4 | Sunken Temple entrance | Use Crescent Key on outer gate |

**Decision-engine rule:** if `Snapshot.Inventory.Contains("Crescent Key") == false && Level >= 48`, engine should route to DM East entrance for Pusillin chase before ST queue.

## Profession Specialty Selection

At skill 200 + level 40, three professions split:

### Blacksmithing (skill 200, level 40)

| Specialty | Trainer | Best for |
|-----------|---------|----------|
| **Armorsmith** | Grumnus Steelshaper (IF) / Okothos Ironrager (Org) | Plate-tank classes (Warrior/Paladin tank); makes Imperial Plate set + endgame Lionheart Helm |
| **Weaponsmith** → Sword/Axe/Mace specialty (skill 250 + level 50) | After Weaponsmith, sub-specialty for one weapon type | DPS-specced classes (Warrior DPS, Paladin Ret, Rogue, Hunter melee) |

### Leatherworking (skill 225, level 40)

| Specialty | Best for |
|-----------|----------|
| **Dragonscale** | Hunter, Shaman Enh — agility/hit gear |
| **Tribal** | Druid Feral, Rogue, Resto Druid — leather caster gear |
| **Elemental** | Caster Druid (Balance), Resto Shaman, agility hybrids |

### Engineering (skill 200, level 30 — earlier than other specs)

| Specialty | Best for |
|-----------|----------|
| **Goblin** | DPS classes; Goblin Sapper Charge AoE damage, Goblin Rocket Boots (PvP/raid speed boost), Goblin Jumper Cables XL (combat resurrect) |
| **Gnomish** | Utility classes; Gnomish Battle Chicken (interrupt), Gnomish Cloaking Device (Vanish-like), Gnomish Death Ray |

**Decision-engine rule:** profession specialty is **account-wide character-locked**. Once selected, switching costs gold + reputation reset. Engine should consult `Snapshot.PreferredCraftingPath` (user-config) before triggering specialty-pickup quest.

## Bracket Exit Conditions

When all of the following are true, engine should mark bracket complete and transition to L50-L60:

- [ ] `Level >= 50`
- [ ] `Talents.PointsSpent == 41` (cumulative)
- [ ] `Profession.Specialty.Selected == true` (BS/LW/Engineering committed)
- [ ] `Dungeons.Cleared.CountOfL40toL50 >= 4` (any 4 of: ZF/Mara3wings/ST/Uldaman-extension)
- [ ] `Hearthstone.Bind == "Thorium Point"` (preferred) OR centrally-located
- [ ] `Profession.Primary.Skill >= 250`
- [ ] `Gold >= 100g` (epic mount fund start: 1000g + 100g target)

If `Snapshot.LevelDelta(start, end) < 10 && Snapshot.WallClock > 16h`, engine flags **bracket stall** — common causes:
- Missing class quest at L50 (Priest/Mage/Druid)
- Profession specialty undecided (BS/LW/Engineering split paralysis)
- Sunken Temple group lockout (Crescent Key not acquired)
- Searing Gorge approach blocked (no FP attune)

## Decision-Engine Rules

1. **At L40 mount turn-in**, dispatch profession-specialty pickup priority (`BS/LW/Engineering split decision`).
2. **Sunken Temple entry rule**: requires Crescent Key + lvl 50+ group. Engine should never queue ST at < lvl 48 (mob density too high) and never without Crescent Key.
3. **ZF Pyramid event**: 4-mob mini-boss event at the top of Zul'Farrak ziggurat ends with **Mosh'aru tablet drops** (ZG attune at 60). Engine should always complete the Pyramid Event even if main quest goal already cleared.
4. **Maraudon entry choice**: orange (Princess wing, lvl 44+ entry) vs purple (outer wing, lvl 42+ entry). Engine should run **purple first** at lvl 42-44 then **orange at 46+** for full bracket coverage.
5. **Cenarion Circle introduction (Druid)**: Pristine Hide quest is the gateway to Silithus rep grind at L60. Engine should pickup at L50 even if Silithus deferred to L60+.
6. **Thorium Brotherhood rep**: from L43 onward, Searing Gorge quests give Thorium Brotherhood rep (capped at Friendly without BRD/MC contribution). Engine should always-pickup all Thorium Point quests on first entry.
7. **Lockpicking 250 gate**: Mara/ST chests require 250+ Lockpicking. Engine should ensure Rogue Lockpicking >= 250 before queuing ST.
8. **Crescent Key chain (Pusillin chase)**: if `Snapshot.Inventory.Contains("Crescent Key") == false && Level >= 48`, route to DM East entrance for Pusillin chase. Pusillin loop is 5-stop; engine should match items at each stop or DPS-rush at final.
9. **Stranglethorn Fishing Extravaganza**: every Sunday 14:00-16:00 server time, neutral fishing event at Booty Bay. Tasty Blue Mussel + Speckled Tastyfish reward chain gives unique rep + recipe drops. Engine should hook event time into wall-clock scheduler.
10. **Gold reserve at L50**: `Gold >= 100g` minimum (start of epic mount savings). Spec into trade-skill flipping (Mining/Herbalism gathered raw → Auction House) for income acceleration.
