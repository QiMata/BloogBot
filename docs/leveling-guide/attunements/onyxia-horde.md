## What it unlocks

Same as Alliance — **Drakefire Amulet** required for Onyxia's Lair entry. The Horde version of the chain is structurally similar but uses different NPCs and storyline (Horde focus on Rexxar's wandering, Eitrigg's elder wisdom, and Thrall's command).

## Comparison to Alliance chain

| Feature | Alliance (Marshal Windsor) | Horde (Eitrigg / Rexxar) |
|---|---|---|
| Quest count | 11 | **14** (more individual steps but each shorter) |
| Starting NPC | Helendis Riverhorn (Burning Steppes) | **Warlord Goretooth** (Kargath, Badlands) |
| Mid-chain set piece | BRD prison rescue + Stormwind masquerade event | **Rend Blackhand UBRS kill** + Rexxar wilderness travel |
| Final UBRS kill | Drakkisath (Blood of Black Dragon Champion) | Drakkisath (Blood of Black Dragon Champion) — same |
| Final reward | Drakefire Amulet | Drakefire Amulet (identical) |
| **Total /played** | ~6-10 hours | **~5-8 hours** (slightly shorter; less escort overhead) |

The Horde chain has **2 UBRS bosses** to kill (Rend Blackhand + Drakkisath) vs Alliance's 1 (Drakkisath only) — but both Horde UBRS kills can happen in a single run.

## Prerequisites

- **Level 60** practical for the UBRS Rend + Drakkisath kills (technically lvl 48 to start at Goretooth, but most chain is gated at 60)
- **Seal of Ascension** for direct UBRS access (or 5-player ritual at door) — see [seal-of-ascension.md](seal-of-ascension.md)
- BRD/LBRS access (Master's Key recommended)
- 10-player UBRS raid group

## Combined UBRS run optimization (Horde-specific)

A Horde UBRS run can complete **3 quest steps simultaneously** if all are accepted before the run:

| Quest | Item drop | Source |
|---|---|---|
| **For The Horde!** | Warchief Rend Blackhand kill | Boss kill |
| **Attunement to the Core** (MC attune) | Drakkisath kill triggers Lothos turn-in flag | Boss kill |
| **Blood of the Black Dragon Champion** (Onyxia attune) | Drakkisath kill drops the Blood item | Boss kill |

**Engine optimization rule (Horde-specific)**: A single UBRS Drakkisath + Rend kill **completes 3 quest steps**: Rend for For The Horde, Drakkisath for both MC and Onyxia attunes. Engine should ensure all 3 quests accepted BEFORE entering UBRS to avoid wasting the 5-day lockout.

## Group requirements

- **Phase 1 LBRS**: 5-player group (bundled with Seal of Ascension chain — see [seal-of-ascension.md](seal-of-ascension.md))
- **Phase 4 UBRS**: 10-player UBRS raid group
- **Mid-chain Rexxar travel + WPL turn-ins**: solo

## Total /played time investment

- **Phase 1 LBRS prep** (bundled with Seal of Ascension): ~1 hour /played extra beyond seal chain
- **Phase 2 Orgrimmar travel + first UBRS for Rend**: ~2-3 hours
- **Phase 3 Rexxar wilderness chain**: ~1-2 hours
- **Phase 4 UBRS Drakkisath**: ~1-2 hours (often same run as Phase 2 if scheduled efficiently)
- **Total**: ~5-8 hours /played, **multi-week** in real time due to UBRS lockout coordination

## VMaNGOS / private server notes

- **Warlord Goretooth in Kargath** is fully scripted on VMaNGOS.
- **Rexxar wandering in Desolace** has historically had pathing issues on some VMaNGOS forks — Rexxar can become stuck on terrain. Engine should plan extended wait time at known patrol points (~5-15 min wait window). `[verify VMaNGOS scripting status pass 3]`
- **Thrall + Eitrigg dialogue** in Orgrimmar is fully scripted.
- **Myranda the Hag in WPL** is correctly placed.
- **Rend Blackhand** in UBRS is the sub-boss before Drakkisath; both kills work correctly.

## Decision-Engine Rules

- **id:** `attune.onyxia-horde.faction-lock` — IF `Faction==Alliance` THEN engine errors out — Horde chain not applicable.
- **id:** `attune.onyxia-horde.lvl-48-start` — IF `Faction==Horde && Level>=48 && !QuestsCompleted.Contains(WarlordsCommand)` AND raid plan includes Onyxia THEN start chain at Warlord Goretooth in Kargath. Priority **750** at lvl 48.
- **id:** `attune.onyxia-horde.bundle-with-seal` — IF `QuestsInProgress.Contains(WarlordsCommand) && QuestsInProgress.Contains(SealOfAscension)` AND running LBRS THEN bundle the LBRS 3-boss kills (same Voone/Omokk/Wyrmthalak targets gate both chains). Priority **820** (logistics optimization).
- **id:** `attune.onyxia-horde.bundle-3-quests-ubrs` — IF `QuestsInProgress.Contains(ForTheHorde) && QuestsInProgress.Contains(AttunementToTheCore) && QuestsInProgress.Contains(BloodOfBlackDragonChampion)` AND UBRS group ready THEN ensure all 3 quests accepted before UBRS run (single run completes Rend + MC + Onyxia attunes). Priority **999** (don't waste UBRS lockout). **Critical optimization (Horde-specific 3-bird-1-stone).**
- **id:** `attune.onyxia-horde.rexxar-wandering` — IF `QuestsInProgress.Contains(ChampionOfTheHorde)` AND in Desolace THEN scan zone for Rexxar (variable spawn point). Priority **700**.
- **id:** `attune.onyxia-horde.amulet-required-for-raid` — IF `RaidScheduledOnyxia && !Items.Contains(DrakefireAmulet)` THEN raid action suppressed. Priority **999** (Onyxia zone-in fails without amulet).
