This file consolidates the **5 major 5-man dungeon keys** that gate access beyond raid attune chains. Each key has its own short quest chain (no raid prereq); the keys are mostly account-character-bound (BoP) so each character must obtain its own.

## Summary table

| Key | Unlocks | Source | Lvl req | Faction | Difficulty |
|---|---|---|---|---|---|
| [Crescent Key](#crescent-key) | Dire Maul N/W/E | Dire Maul East — chase Pusillin imp | 55+ | both | Medium (chase + cornered fight) |
| [Shadowforge Key](#shadowforge-key) | BRD beyond first wing (Grim Guzzler, Imperial Hall) | BRD — Dark Iron Legacy quest from Franclorn Forgewright ghost | 52+ | both | Medium (Fineous Darkvire kill in BRD) |
| [Master's Key](#masters-key) | BRM orb between LBRS / UBRS / MC | BRD lower city `[verify pass 3]` | 55+ | both | Medium |
| [Workshop Key](#workshop-key) | Gnomeregan Workshop wing | Tinker Town (Ironforge) / Steelgrill's Depot quest chain | 25-29 | Alliance-leaning | Easy |
| [Skeleton Key](#skeleton-key) | Scholomance | Chillwind Camp (WPL) — Scholomance quest chain | 55+ | both | Medium (multi-step Andorhal + Thorium forging) |

**Engine note**: Of these 5, only **Skeleton Key** and **Shadowforge Key** are absolutely raid-relevant (Scholo and BRD are part of pre-raid + class-quest chains). The others are convenience keys — Crescent is required for DM tribute Quel'Serrar (Warrior class chain), Master's is logistics-only, Workshop is leveling-bracket utility.

## Crescent Key

**Unlocks**: Dire Maul (all three wings — North, West, East). Without the Crescent Key, only DM East is enterable through the side passage; West and North are locked behind doors that the key opens.

### Acquisition

| Step | Location | Goal |
|---|---|---|
| 1 | **Dire Maul East entrance** (no key required) — enter from Feralas | Reach the inside of DM E |
| 2 | Locate **Pusillin** the imp (small imp NPC patrolling DM East) | Approach Pusillin — he will run away when engaged |
| 3 | **Chase Pusillin** through DM East — he flees to multiple checkpoint locations, stopping at each before running again | The chase lasts ~5-10 minutes through several rooms |
| 4 | When Pusillin reaches his **final checkpoint** (a chamber near the back of DM E), engage him in combat | Pusillin is a lvl 60 elite imp — solo for stealth/Mage/Druid; 2-3 group otherwise |
| 5 | Loot **Crescent Key** from his corpse | BoP, soulbound to looter |

**Tactics**: Druid Cat Form / Rogue Stealth makes the chase trivial. Mages can Frost Nova → Blink past mob packs. The chase doesn't aggro most of DM E if done carefully.

**Engine planning**: Bundle the Pusillin chase with a planned DM E run for other objectives (Foror's Compendium for Hunters, Lethtendris recipe drops, Alzzin the Wildshaper kills for tribute prep).

## Shadowforge Key

**Unlocks**: Blackrock Depths beyond the first wing — specifically the **Grim Guzzler tavern** door and the path to **Imperial Hall / Emperor Dagran Thaurissan**. Without the Shadowforge Key, parties must take a Rogue pickpocket / lockpick to bypass these doors.

### Acquisition — Dark Iron Legacy chain

| Step | Location | Goal |
|---|---|---|
| 1 | **Inside BRD** — find the **tomb** with **Franclorn Forgewright**'s ghost atop it (located in the BRD upper area near the Grim Guzzler) | Player must **be dead/ghost** to interact with Franclorn |
| 2 | Talk to **Franclorn Forgewright** (ghost) | Receive quest **Dark Iron Legacy** |
| 3 | Kill **Fineous Darkvire** (BRD boss) | Loot **Ironfel** (a special hammer) |
| 4 | Find the **statue of Franclorn Forgewright** elsewhere in BRD | Use Ironfel on the statue (right-click) |
| 5 | Quest auto-completes; receive **Shadowforge Key** | BoP |

**Group requirements**: Standard 5-player BRD group; Fineous Darkvire is a regular BRD boss along the Imperial Hall path.

**Engine planning**: This chain is **highly recommended for any raid character** — Shadowforge Key access in BRD enables fast Bar runs (Hand of Justice trinket farm, Spectral Chalice quest for Mining recipe, Phalanx kill for Ironfoe drop attempts). Bundle with planned BRD runs.

## Master's Key

**Unlocks**: The **Black Iron Refinery / BRM orb access** between Blackrock Mountain entrance lobby and BRD/LBRS/UBRS/MC instance portals. Without Master's Key, players walk the long ramps. With it, an orb teleport accelerates travel.

## Workshop Key

**Unlocks**: The **Workshop wing of Gnomeregan** (the area beyond Mekgineer Thermaplugg's blocked door — the irradiated workshop area where engineering recipe drops appear).

### Acquisition

| Step | Location | Goal |
|---|---|---|
| 1 | **Tinker Town (Ironforge)** or **Steelgrill's Depot (Dun Morogh)** — pick up the introductory Gnomeregan quest from a Gnome questgiver `[verify pass 3]` | Quest start |
| 2 | Run **Gnomeregan** (lvl 29-38 dungeon) | Standard 5-man |
| 3 | Loot the **Workshop Key** from a specific mob in Gnomeregan (often a goblin / mechano-hostile NPC; `[verify pass 3 exact mob]`) | BoP |

**Engine planning**: Lvl 25-35 bracket — pick up during normal Gnomeregan runs. Not raid-relevant; affects only Gnomeregan workshop access for engineering recipe farming.

## Skeleton Key

**Unlocks**: **Scholomance** dungeon entrance.

### Acquisition — Scholomance quest chain

| Step | NPC / Location | Goal |
|---|---|---|
| 1 | Faction-specific quest giver: **Commander Ashlam Valorfist** (Alliance, Chillwind Camp, WPL) OR **High Executor Derrington** (Horde, Chillwind Camp, WPL) | Receive quest **Scholomance** |
| 2 | Travel to **Andorhal** (Western Plaguelands ruined city) | Kill reanimated skeleton mobs to loot **bone fragments** (specific quest items) |
| 3 | Return to Chillwind Camp questgiver | Hand over fragments; receive imbued items + Thorium Bar reagent requirement |
| 4 | Acquire **Thorium Bar** (Mining or AH) for the key blank | 1× Thorium Bar |
| 5 | Forge the **Skeleton Key** at the questgiver / quest event | BoP |

**Group requirements**: Andorhal mob kills are solo-viable for a lvl 58+ character; the Scholomance entry test itself requires a 5-player group.

**Engine planning**: Skeleton Key is **mandatory** for several lvl-60 class chains:

- **Paladin Charger** (Phase 5: Death Knight Darkreaver in Scholomance — see [paladin.md](../classes/paladin.md))
- **Warlock Dreadsteed** (Phase 4: Kanrethad summoning ritual in Scholomance — see [warlock.md](../classes/warlock.md))
- **Priest Benediction** chain — requires Scholo step `[verify pass 3]`
- General Scholomance gear farming + Headmaster's Charge staff farming for caster classes

Engine should plan Skeleton Key acquisition **before any of the above class chains** are scheduled.

## Decision-Engine Rules

- **id:** `key.crescent.acquire` — IF `Level>=55 && !KeysInBags.Contains(CrescentKey)` AND DM E run scheduled THEN chase Pusillin + loot key. Priority **750** when bundled with other DM E objectives.
- **id:** `key.shadowforge.acquire` — IF `Level>=52 && !KeysInBags.Contains(ShadowforgeKey)` AND BRD run scheduled THEN run Dark Iron Legacy chain (find Franclorn ghost → kill Fineous Darkvire → use Ironfel on statue). Priority **820** (raid-prep utility — gates BRD inner access).
- **id:** `key.shadowforge.die-to-talk-franclorn` — IF `QuestsInProgress.Contains(DarkIronLegacy_init) && !FranclornGhostTalkedTo && InBRD` THEN suicide briefly to ghost-walk to Franclorn (or wait for natural death). Priority **decision-request** (intentional death is unusual; engine confirms).
- **id:** `key.skeleton.acquire-before-class-chain` — IF `(Class==Paladin && PlannedChargerChain) OR (Class==Warlock && PlannedDreadsteedChain) OR (Class==Priest && PlannedBenedictionChain)` AND `!KeysInBags.Contains(SkeletonKey)` THEN run Chillwind Camp + Andorhal Skeleton Key chain BEFORE class chain. Priority **820**.
- **id:** `key.workshop.acquire` — IF `Level IN [25,38] && Class.Engineering OR PlannedGnomereganRun && !KeysInBags.Contains(WorkshopKey)` THEN bundle with Gnomeregan run. Priority **400** (background; not raid-critical).
- **id:** `key.master.optional` — IF `Level>=55 && !KeysInBags.Contains(MastersKey) && BRMLobbyVisitsThisWeek > 5` THEN consider acquiring Master's Key for ramp-walk savings. Priority **300** (optional convenience).
