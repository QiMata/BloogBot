## What it unlocks

The **Seal of Ascension** is a +10 Fire/Frost/Nature resist ring that allows direct entry to **Upper Blackrock Spire (UBRS)** without requiring a 5-player ritual at the door. Without the Seal, UBRS access requires **5 players present at the door** to perform a summoning ritual — viable but slower for repeat runs.

UBRS itself is the **central hub of vanilla raid attunement**:

| UBRS unlocks | Why it matters |
|---|---|
| **MC attunement** | Kill Drakkisath → loot **Drakkisath's Brand** → turn in to **Lothos Riftwaker** in BRM lobby for MC teleport access |
| **BWL attunement** | Talk to **Vaelastrasz** orb in BRS upper (in Vael's chamber inside BWL portal area) for the BWL portal flag |
| **Onyxia attunement** (both factions) | Multi-step chain requires a UBRS Drakkisath kill (Alliance: head to Marshal Windsor; Horde: Warchief's Mandate) |
| **Pre-raid BiS gear** | Drakkisath, Solakar Flamewreath, Pyroguard Emberseer drops |
| **Tier 0 helmets** for several classes (Drakkisath, Pyroguard) | T0 piece source |

**The Seal is therefore the single highest-value attunement-for-attunement-prerequisite chain in 1.12.1** — it's required *before* MC, BWL, and Onyxia attunes can begin.

## Prerequisites

- Level 60 strongly recommended (chain steps require LBRS + UBRS + Dustwallow Marsh and elite mob kills; possible at 58+ with a strong group)
- **LBRS access** (no key required — entrance via Master's Key OR walking through BRM)
- A 5-player group capable of running LBRS (which itself unlocks via the BRM orb / Master's Key)

## Chain step-by-step

### Phase 1 — Find Vaelan in LBRS (the disguised quest giver)

| Step | NPC / Location | Item / Goal | Notes |
|---|---|---|---|
| 1.1 | Run **LBRS** (no key required for LBRS itself) | Loot **Unadorned Seal of Ascension** (drops from various LBRS mobs, especially Smolderthorn/Bloodaxe/Spirestone trash) | Common drop; usually obtained during a normal LBRS run |
| 1.2 | **Vaelan** in LBRS (disguised as **Scarshield Infiltrator** until you walk up to him; located after climbing the stone ramp upward in the early section) | Approach Vaelan (he reveals his true form) | First trip; many groups don't notice him on a first LBRS run. Engine should plan a deliberate approach. |
| 1.3 | Talk to Vaelan with the Unadorned Seal in inventory | Receive quest **"Seal of Ascension"** | Quest start NPC for the chain |

### Phase 2 — Collect 3 Gemstones of Command

Run LBRS (1-2 full clears typically, since the 3 boss drops aren't always all on one lockout). The 3 gemstones drop from named LBRS bosses:

| Gemstone | Drops from | Boss location |
|---|---|---|
| **Gemstone of Smolderthorn** | **War Master Voone** | Hall of Blackhand area; mid-LBRS |
| **Gemstone of Spirestone** | **Highlord Omokk** | First main boss; left-side path |
| **Gemstone of Bloodaxe** | **Overlord Wyrmthalak** | Late-LBRS final boss before exit to Burning Steppes side |

**Drop rates**: 100% guaranteed drop from each respective boss `[verify pass 3 — typically 100% on VMaNGOS for attunement-flag drops]`.

**Engine planning**: A single LBRS run usually yields 2-3 of the 3 gemstones; second run completes. Bundle with regular T0 farming runs.

### Phase 3 — Forge the Seal at Emberstrife in Dustwallow Marsh

Once all 3 gemstones + Unadorned Seal are in inventory:

| Step | NPC / Location | Item / Goal | Notes |
|---|---|---|---|
| 3.1 | Return to Vaelan in LBRS with all 4 items | Vaelan gives quest **"What the Wind Carries"** — directs you to Dustwallow Marsh | |
| 3.2 | **Dustwallow Marsh** — find the dragon **Emberstrife** in his cave (~75, 50 area `[verify pass 3]`) | Open dialogue with Emberstrife | Emberstrife is a lvl-60 elite dragon; **do NOT kill** — subjugation only |
| 3.3 | Use **Orb of Dragonic Energy** (quest-given) on Emberstrife | Subjugate the dragon; Emberstrife forges the **Forged Seal of Ascension** | Group not strictly required for the subjugation step (it's a scripted dialogue interaction); however **Emberstrife will fight back** if approached without proper precaution. Engine plans 2-3 player group as safety. |
| 3.4 | Return to Vaelan in LBRS with the **Forged Seal of Ascension** | Final hand-in | |
| 3.5 | Vaelan gives the **Seal of Ascension** ring | Wear or carry in bags — used as a key item to enter UBRS | The Seal is BoP and **does not consume on use** — one-time chain reward |

### Reward stats: Seal of Ascension

- **+10 Fire Resistance**
- **+10 Frost Resistance**
- **+10 Nature Resistance**
- Unique: required to enter UBRS without a 5-player ritual

The +10 tri-resist is mid-tier and not BiS — many classes have better rings by this point. **The Seal's value is the UBRS access, not the stats.** Players often equip it during travel-to-UBRS and swap to a BiS ring before pulls.

## Group requirements

- **LBRS phase**: Standard 5-player group (tank + healer + 3 DPS)
- **Emberstrife phase**: Solo to 2-player. Subjugation is a dialogue interaction; Emberstrife resets without aggro escalation typically. Some groups bring 1 extra DPS as a safety net for accidental aggro.

## Total /played time investment

- **Best case**: 1-2 LBRS runs (~90-120 minutes) + 1 Dustwallow trip (~30 minutes) = **~2-3 hours** total /played
- **Average**: 2-3 LBRS runs needed to drop all 3 gemstones + Dustwallow trip = **~3-5 hours**
- **Bundling with T0 farming**: Engine often slots Seal of Ascension chain into the same lockouts as T0 farming for the character; the gemstones are a side-loot

## VMaNGOS / private server notes

- **Vaelan** in LBRS is fully scripted and reveals on approach correctly.
- **Gemstones** drop reliably from all 3 LBRS bosses on VMaNGOS.
- **Emberstrife subjugation** is the one scripting wrinkle on some private servers — the Orb of Dragonic Energy interaction can fail if Emberstrife is engaged in combat. Engine should plan to approach **Emberstrife not in combat with anything else**. `[verify VMaNGOS scripting status pass 3]`
- **Drakkisath quest item (Drakkisath's Brand)** is the next attunement step (MC) — drops 100% from Drakkisath inside UBRS.

## Decision-Engine Rules

- **id:** `attune.ubrs.collect-unadorned-seal` — IF `Level>=58 && !Items.Contains(UnadornedSealOfAscension)` AND running LBRS THEN passively collect during run. Priority **750** (always loot trash drops).
- **id:** `attune.ubrs.find-vaelan` — IF `Level>=58 && Items.Contains(UnadornedSealOfAscension) && !QuestsCompleted.Contains(SealOfAscension_step1)` AND in LBRS THEN approach Vaelan (disguised NPC, requires deliberate proximity). Priority **800**.
- **id:** `attune.ubrs.collect-gemstones` — IF `Level>=58 && QuestsInProgress.Contains(SealOfAscension) && Items.GemstoneCount<3` AND running LBRS THEN prioritize killing the 3 named bosses (Voone, Omokk, Wyrmthalak). Priority **820** (bundle with normal LBRS clear).
- **id:** `attune.ubrs.emberstrife-subjugate` — IF `Items.Contains(All3Gemstones) && Items.Contains(UnadornedSeal) && QuestsInProgress.Contains(WhatTheWindCarries)` THEN travel to Dustwallow Marsh + subjugate Emberstrife. Priority **820** (single-trip completion). **Group of 2-3 recommended for safety.**
- **id:** `attune.ubrs.completion-prereq-mc` — IF `!Attunements.Contains(MoltenCore) && !Items.Contains(SealOfAscension)` AND raid plan includes MC THEN engine plans Seal of Ascension chain BEFORE MC attune chain (since MC attune requires UBRS access for Drakkisath's Brand). Priority **820**.
- **id:** `attune.ubrs.completion-prereq-bwl` — IF `!Attunements.Contains(BlackwingLair) && Level==60` AND raid plan includes BWL THEN engine plans Seal of Ascension → UBRS clear → Vaelastrasz orb in BRS upper. Priority **820**.
