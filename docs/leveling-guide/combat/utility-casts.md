# Utility Casts — Combat-Only Cooldown Ladder → CC Application → Curse Priority → Class-Specific Utility → Folded Rest-Rotation OOC Recovery → Drink-to-95 / Eat-to-95 → Buff-Refresh Tick on Rest → Snapshot Contract → Cancellation Sequencing

Closes the Combat sub-family. Where [melee-rotation.md](./melee-rotation.md) drives the swing/cooldown ladder, [cast-spell.md](./cast-spell.md) drives any single cast, [pull-target.md](./pull-target.md) drives pre-engage, [threat-management.md](./threat-management.md) governs who the mob chases, [heal-task.md](./heal-task.md) governs heal-target selection, and [buff-cycle.md](./buff-cycle.md) governs the always-on aura prologue, **utility-casts** is the residual surface containing everything else a combat-aware bot does that is **NOT** an auto-swing, a damage cast, a heal, or a prologue buff: (a) the **combat-only utility cooldown ladder** (Inner Focus, Power Infusion, Combustion, Adrenaline Rush, Blade Flurry, Death Wish, Bloodrage, Berserker Rage, Evasion, Riposte, BloodFury, Retaliation); (b) **CC application** (Polymorph, Sap, Hibernate, Banish, Fear, Wyvern Sting — distinct from rotation damage); (c) **curse priority** (Curse of Elements / Tongues / Recklessness / Agony / Shadow); (d) **class-specific situational utility** (Tremor Totem fear-break, Innervate combat-MP donation, Lay on Hands emergency-full-HP, Vanish + Feign Death threat-wipes); (e) the **folded rest-rotation OOC counterpart** — drink-to-95 / eat-to-95 between pulls, buff-refresh tick on rest, mount-back-up after rest, combat-interrupt on rest. This doc picks up where [buff-cycle.md R13](./buff-cycle.md) defers (combat-only utility cooldowns explicitly excluded from `BuffTask` — Inner Focus, Power Infusion, Combustion, Adrenaline Rush, Blade Flurry, Death Wish, Bloodrage, Berserker Rage all belong here), where [melee-rotation.md R15 + R17](./melee-rotation.md) defer (post-combat `RestTask` push gate at HP<95 / mana<80 + per-spec RestTask body), and where [threat-management.md R8](./threat-management.md) defers (Vanish / Feign Death threat-drop mechanics — `HostileRefManager` aggro-wipe).

## Overview

Utility casts in 1.12.1 split into **four families**, each with distinct server-side mechanics and bot-side dispatch patterns:

**Family 1 — Combat-only self-buff cooldowns** (Inner Focus / Power Infusion / Combustion / Adrenaline Rush / Blade Flurry / Death Wish / Bloodrage / Berserker Rage / Retaliation / Evasion). These apply a temporary buff aura on self (1-30 s duration) that boosts damage / mana / crit / armor / etc. Cooldowns 1-5 min. Dispatched via `CastSpell(name, castOnSelf: true)` or `TryUseAbility(name, cost, condition)`. Wire-side `CMSG_CAST_SPELL` with `TARGET_FLAG_SELF = 0x0000` per [cast-spell.md R14](./cast-spell.md). **Belong in `PvERotationTask` (combat-tick), NOT in `BuffTask`** — gated on `IsInCombat == true && IsSpellReady(name) && target.HealthPercent > 80` (most damage-amplifier cooldowns front-load on the opening).

**Family 2 — CC application** (Polymorph / Sap / Hibernate / Banish / Fear / Wyvern Sting / Scare Beast / Repentance / Cyclone-TBC). Server-side: `SPELL_AURA_MOD_CONFUSE = 5` (`SpellAuraDefines.h:63`), `SPELL_AURA_MOD_FEAR = 7` (`SpellAuraDefines.h:65`), `SPELL_AURA_MOD_PACIFY = 25` (`SpellAuraDefines.h:156`), `SPELL_AURA_MOD_PACIFY_SILENCE = 60` (`SpellAuraDefines.h:198`). All are `TARGET_FLAG_UNIT` ranged casts on a hostile non-target (the secondary aggressor or pre-pull engagement bypass). Sap is stealth-gated (`SPELL_AURA_MOD_STEALTH` precondition). Polymorph/Sap break on damage; Hibernate breaks on damage. **Open SCo.1** — no shipped profile dispatches CC today; the open work is to add a `TryCastCcOnSecondaryAggressor` helper layered above `Aggressors.Where(a => a.Guid != target.Guid)` selection.

**Family 3 — Curse priority (Warlock)**. The 5 1.12.1 Warlock curses are `Curse of Agony` (DoT damage — high single-target DPS contribution; shipped in `Spellbook.cs:38`), `Curse of Recklessness` (-armor + fear-immune — used to ANTI-fear bosses), `Curse of Tongues` (-50% cast-speed — lockdown of caster mobs), `Curse of Elements` (-resist + amp damage — caster party support), `Curse of Shadow` (-shadow resist — Warlock-DPS amp). **Mutex** — only ONE Warlock curse per target at a time. Curse priority: caster-party context prefers Elements (caster DPS amp) or Tongues (caster mob lockdown); melee-party context prefers Recklessness (-armor for melee DPS); solo prefers Agony (Warlock's own DPS contribution). The shipped surface is partial — `CurseOfAgony` ships in BotRunner Spellbook ([`Exports/BotRunner/Constants/Spellbook.cs:45`](../../../Exports/BotRunner/Constants/Spellbook.cs)) but `CurseOfElements`, `CurseOfTongues`, `CurseOfRecklessness`, `CurseOfShadow` are **NOT** in `Spellbook.cs` (open SCo.1 Spellbook gap, mirror of [buff-cycle.md cross-ref note](./buff-cycle.md)).

**Family 4 — Class-specific situational utility** (Innervate / Lay on Hands / Vanish / Feign Death / Tremor Totem / Grounding Totem / Cleanse / Dispel / Counterspell / Spell Reflection). Each has unique server-side mechanics — see [#Decision-Engine Rules](#decision-engine-rules) for per-spell threshold tables.

**Rest-rotation (the OOC counterpart, folded)** uses the same `IObjectManager` reads + `Wait.For` debounce pattern as the inline RestTask bodies. Pop-predicate uniformity is the canonical R1 trigger; recovery actions are per-spec but the lifecycle is uniform — bandage-first if HP<60 + no debuff, then eat best food, then drink (mana-classes only). On Pop, **mana-class RestTask pushes BuffTask** (MageFrost:28) to refresh expiring auras (the buff-refresh tick from [buff-cycle.md R6](./buff-cycle.md) — this is the explicit cross-doc fold-in). Physical-class RestTask does NOT push BuffTask on Pop (no shipped BuffTask body for WarriorFury — the warrior buff prologue is inline in `PvERotationTask` via `BattleShout`).
