# Stranglethorn Vale (STV) — Contested L30-45 Zone (Hemet Nesingwary, ZG Raid, Fishing Tournament)

Eastern Kingdoms tropical jungle zone — **The largest single-zone bracket** spanning L30-45 in vanilla 1.12.1. Sub-zones: **Booty Bay** (neutral goblin port, **primary cross-faction hub**), **Grom'gol Base Camp** (Horde hub), **Rebel Camp** (Alliance hub), **Nesingwary's Expedition** (Hemet's base), **Yojamba Isle** (Zandalar Tribe NPCs), **Mosh'Ogg Ogre Mound** (L35-39), **Bal'lal Ruins** (trolls), **Ruins of Aboraz / Jubuwal** (Zul'Gurub raid area). **Zul'Gurub** raid (lvl 60 20-man) at southern tip. Iconic content: **Hemet Nesingwary Big Game Hunter chain**, **STV Fishing Extravaganza Sunday tournament**, **Bloodsail vs Steamwheedle Cartel rep mutual exclusion**. The bracket-defining cross-continent transition zone.

## Hemet Nesingwary's Big Game Hunter Chain (Iconic L30-40)

**The bracket-defining XP windfall chain**.

| Step | Action | Reward |
|------|--------|--------|
| 1 | Pickup at Nesingwary's Expedition | Initial chain |
| 2 | **Stranglethorn Tigers** (kill multiple, ~25-30) | Kill quests |
| 3 | **Panthers** (kill multiple) | Kill quests |
| 4 | **Raptors** (kill multiple) | Kill quests |
| 5 | **King Bangalash** (rare tiger elite, party-pull) | Trophy quest |
| 6 | **The Hunter** (final cross-quest) | Major XP windfall |
| 7 | **Hunter quest set** (Hunter-only quest set, **Hunter class quest helm**) | Hunter BiS-mid-tier helm |

**Decision-engine rule:** Hemet Nesingwary chain is **the** XP windfall for L30-40. Engine should always-pickup at L30 entry. **Hunter alts** specifically benefit from class-quest helm reward.

## STV Fishing Extravaganza (Sundays)

The **Stranglethorn Fishing Extravaganza** is a weekly tournament at Booty Bay.

| Detail | Value |
|--------|-------|
| Schedule | Sunday 14:00-16:00 **server time** |
| Goal | Catch 40 Speckled Tastyfish from STV pools, deliver to **Riggle Bassbait** at Booty Bay |
| 1st place | **1000g** + **Title: "Master Angler of Stranglethorn"** + **Arcanite Fishin' Pole** (+35 skill, BiS) |
| 2nd-4th | **Hook of the Master Angler** trinket; **Nat Pagle's Extreme Angler FC-5000** (+25 skill); **Lucky Fishing Hat** (+5 skill) |
| Random | Consolation rewards for participants |

See [../professions/fishing.md](../professions/fishing.md#stranglethorn-fishing-extravaganza-sundays) for full event detail.

## Bloodsail Buccaneers vs Steamwheedle Cartel (Mutual Exclusion)

**Choosing Bloodsail rep makes Booty Bay Hostile**.

| Path | Pros | Cons |
|------|------|------|
| **Steamwheedle (default)** | Engineer recipes; AH access; standard raiding | No Pirate cosmetic set |
| **Bloodsail (opt-in)** | Pirate cosmetic 4-piece set; ZG cosmetic | Booty Bay Hostile = no AH/recipes/quests at any Steamwheedle port |

**Decision-engine rule:** at L30+ in STV, default to **Steamwheedle path**. Bloodsail opt-in cosmetic only.

See [../reputations/steamwheedle-cartel.md](../reputations/steamwheedle-cartel.md) for full mutual-exclusion detail.

## Decision-Engine Rules

1. **Bracket entry**: STV is the **L30-45 contested zone + cross-continent boat hub**. Engine should plan ~15-20 hour completion through L40.
2. **Hearthstone bind**: Booty Bay Inn is the cross-faction neutral default. **Mandatory L30+ rebind** for STV-Tanaris-Hinterlands axis.
3. **Hemet Nesingwary chain priority**: bracket-defining XP windfall + Hunter class quest helm. Engine should always-pickup at L30 entry.
4. **Bloodsail vs Steamwheedle decision**: default Steamwheedle (raid path); Bloodsail opt-in cosmetic only.
5. **STV Fishing Extravaganza tracking**: Sunday 14:00-16:00 server time. Engine should mark calendar.
6. **ZG raid prep at southern STV**: defer to L60 raid week. Engine should track Yojamba Isle NPCs for Zandalar Tribe rep.
7. **Cross-zone Ratchet boat**: fastest cross-continent for Alliance + Horde. Engine should plan boat-timing.
8. **Mosh'Ogg ogre area**: lvl 35-39; engine should party-pull or solo at L37+.
9. **Profession early grind**: Booty Bay sufficient for advanced professions at L30-40. Capital cities for specialty.
