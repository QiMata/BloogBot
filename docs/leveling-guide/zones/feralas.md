# Feralas — Contested L40-50 Zone (Dire Maul, Sprite Darter Rescue, Pusillin Chase)

Kalimdor jungle zone — **Contested** with major hubs for both factions. Sub-zones: **Feathermoon Stronghold** (Alliance hub on Sardor Isle, accessed via hippogryph from Thalanaar), **Camp Mojache** (Horde hub), **Thalanaar** (neutral entry FP hub), **Dire Maul ruins** (entrance to **3-wing L55-60 dungeon** — Pusillin chase entrance area gives Crescent Key), **Sprite Darter Pen** (Sprite Darter rescue area), **Forgotten Coast** (south naga — Lord Shalzaru), **Hippogryph Pens** (protection), **Bristlemane Hold**. Notable mobs: **Gordunni ogres** (Dire Maul), **Wildkin/Owlkin**, **Hippogryph** (protected), **Sprite Darters** (Kindal Moonweaver rescue), **Grimtotem Tauren** (Sprite Darter captors), **Lord Shalzaru** naga. **Pusillin chase** at DM East entrance — Crescent Key for Sunken Temple.

## Decision-Engine Rules

1. **Bracket entry**: Feralas is the **L40-50 contested zone**. Engine should plan ~10-15 hour completion through L48.
2. **Hearthstone bind**: Feathermoon (A) / Camp Mojache (H) Inn is the bracket-5 default.
3. **Pusillin Crescent Key priority**: at L48+, ALWAYS pickup before Sunken Temple queue. Engine should encode 5-stop chase pattern or DPS-rush at L50+.
4. **Sprite Darter rescue**: Sprite Darter Hatchling vanity pet — Cenarion-aligned quest. Engine should pickup for completionist alts.
5. **Dire Maul dungeon prep**: at L55+, plan group queue. DM-N (Tribute Run) + DM-W (Pylon mechanic) + DM-E (Pusillin chase + Felvine Shard) — see dungeon files.
6. **Thalanaar hippogryph**: mandatory for Alliance Feathermoon access. Engine should encode FP route.
7. **Cenarion Circle parallel rep**: Camp Mojache + Feathermoon both have Cenarion-aligned quests. Engine should pickup at L40+ for L55+ Silithus prep.
8. **Profession early grind**: Feathermoon/Camp Mojache sufficient for advanced primary professions at L40-50.
