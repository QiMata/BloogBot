# Ashenvale — Contested L18-30 Zone (BFD Dungeon Entry, WSG Entry)

Kalimdor forest zone — **Contested** between Alliance (Sentinels) and Horde (Warsong Outriders). Sub-zones: **Astranaar** (Alliance main hub), **Splintertree Post** (Horde main hub), **Maestra's Post** (Alliance, north), **Hellscream's Watch** (Horde sub-east), **Zoram'gar Outpost** (Horde, far NW coast — BFD dungeon adjacent), **Forest Song** (Draenei outpost), **Felfire Hill** (demon-corrupted), **Xavian** (satyr camp), **Bough Shadow** (Emerald Dragon Taerar spawn), **Warsong Lumber Camps** (3 contested logging camps), **Silverwing Grove** (Alliance WSG entry). **Blackfathom Deeps** dungeon (lvl 20-30) at Zoram Strand. Notable mobs: **Demon** (Felfire/Xavian), **Burning Blade clan**, **Corrupted Furbolgs** (Bloodvenom Post), **Naga**, **Dragonspawn** (Bathran's Haunt).

## Decision-Engine Rules

1. **Bracket entry**: Ashenvale is the **L18-30 contested zone**. Engine should plan ~10-15 hour completion through L25.
2. **Hearthstone bind**: Astranaar (A) / Splintertree Post (H) Inn is the bracket-3 default.
3. **WSG entry priority**: at L20+, queue WSG via Silverwing Grove (A) or Warsong Lumber Camp (H). Cross-BG honor accumulation.
4. **BFD dungeon prep**: at L20-25, plan group queue for Blackfathom Deeps via Zoram'gar Outpost. Je'neu Sancrea pre-quest is mandatory.
5. **Raene Wolfrunner Furbolg chain**: long Alliance multi-zone chain extends to Felwood L48+. Engine should pickup early but defer completion.
6. **Bough Shadow world boss (Taerar)**: defer to L60 raid week. Engine should track for raid coordination.
7. **Warsong Lumber Camps PvP-flagging**: cross-faction lumber camp quests can flag PvP. Engine should ensure group support.
8. **Profession early grind**: Astranaar/Splintertree sufficient for advanced professions at L20-30. Capital cities for specialty.
