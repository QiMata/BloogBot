# Mulgore — Tauren Starter Zone (L1-10)

Kalimdor central plains zone — **Tauren-only Horde starter zone**. Sub-zones: **Red Cloud Mesa** (Lvl 1-5 enclosed starter), **Camp Narache** (L1-5 starter hub), **Bloodhoof Village** (L5-10 main hub), **The Golden Plains**, **Wildmane Water Well**, **Brave's Rise**, **Ravaged Caravan**. Capital: **Thunder Bluff** (atop the central mesa, accessed via tauren-built ramp from Bloodhoof Village). Notable mobs: **Plainstrider**, **Mountain Cougar**, **Wildmane Furbolg**, **Bristleback Quilboar**, **Mottled Boars**, **Prairie Wolves**. Class identity quests: **Shaman Earth + Fire Totem** (Tauren racial), **Druid Bear Form Moonglade** (Tauren racial), Hunter L10. Tauren classes are limited (Druid/Hunter/Shaman/Warrior — NO Priest/Mage/Warlock/Rogue/Paladin).

**Enclosed Tauren-only starter area** — first 5 levels.

- **Rite of Strength** — kill Plainstrider for tribal initiation
- **Rite of Vision** — vision quest mid-tier

**Decision-engine cue:** Red Cloud Mesa is the **enclosed L1-5 Tauren starter**. Engine should plan ~1-1.5 hour completion through to L5 + Camp Narache arrival.

## Class Quests (L1-10)

### Tauren Class Restrictions

Tauren can only be: **Druid, Hunter, Shaman, Warrior**. Cannot be Priest/Mage/Warlock/Rogue/Paladin.

### Shaman (Earth + Fire Totem L4-10 Chain)

| Step | Source NPC | Reward |
|------|-----------|--------|
| 1 | Camp Narache Shaman trainer | First Shaman racial quest |
| 2 | Various Camp Narache + Bloodhoof quests | Earth Totem at L4-5 |
| 3 | Travel to Thunder Bluff Shaman trainer (Hunter Rise) | Fire Totem chain at L10 (Mountain trip in Mulgore) |
| 4 | Defeat Minor Manifestation of Fire | Fire Totem trained |

**Decision-engine rule:** Shaman is **Tauren racial**; engine should pre-flag for Tauren Shaman alts.

### Druid (Bear Form Moonglade Chain)

| Step | Source NPC | Reward |
|------|-----------|--------|
| 1 | Druid trainer in Bloodhoof Village | "Cleansed Spirit" mid-quest |
| 2 | Travel to **Moonglade** (port from Druid trainer) | Free port at L10+ |
| 3 | Defeat **Lunaclaw** (lvl 11 elite) | Soul of Lunaclaw item |
| 4 | Return to trainer | **Bear Form** spell trained |

**Decision-engine rule:** Druid Bear Form is **the** L10 Tauren class quest (mirrors NE Druid). Engine should plan Moonglade portal route + Lunaclaw kill.

### Hunter (Tame Beast L10)

| Step | Source NPC | Reward |
|------|-----------|--------|
| 1 | Hunter trainer in Bloodhoof Village / Thunder Bluff | Tame Beast quest chain |
| 2 | Defeat 3 race-specific beasts (Plainstrider, Mountain Cougar, Strider variant) | Tame Beast spell trained |

**Decision-engine rule:** at lvl 10, Tauren character should travel to Thunder Bluff for class trainer + class quest. Engine should plan Bloodhoof→Thunder Bluff ramp path.

- **All primary professions** + secondary

**Decision-engine rule:** Bloodhoof Village sufficient for L1-15. Thunder Bluff for L20+ + specialty.

## Decision-Engine Rules

1. **Starter zone gate**: Red Cloud Mesa (L1-5) → Camp Narache → Bloodhoof Village (L5-10) → Thunder Bluff (class quests at L10) → Barrens (L10-25 transition).
2. **Hearthstone bind**: Bloodhoof Village Inn (Innkeeper Kauth) is the bracket-1 default. Engine should bind here after first quest from Bloodhoof.
3. **Class restrictions**: Tauren can only be Druid/Hunter/Shaman/Warrior. Engine should fail-fast for invalid Tauren-class character creation.
4. **Druid Bear Form chain**: Tauren Druid mirrors NE chain (Moonglade port + Lunaclaw kill). Engine should plan portal route from Bloodhoof Druid trainer.
5. **Shaman Earth + Fire Totem chain**: Tauren Shaman racial. Engine should plan Camp Narache → Bloodhoof → Thunder Bluff → Mountain trip (Telf Joolam-equivalent in Mulgore mountains).
6. **Class quest dispatch (L10)**: Hunter must travel to Thunder Bluff (Holt Thunderhorn). Engine should plan Bloodhoof→Thunder Bluff ramp path.
7. **Barrens transition**: at L8-10, head east to Crossroads via Bloodhoof east road. Standard route.
8. **Profession early grind**: Bloodhoof Village sufficient for L1-15. Thunder Bluff for advanced.
9. **Princess Mina + Skull Crusher elite**: Brave's Rise quilboar elites. Engine should party-pull or solo at L11+.
10. **Capital ramp route**: Bloodhoof→Thunder Bluff ramp is the Tauren shortcut to capital. Engine should plan early visit.

```text
Snapshot.Level                                    // bracket gate
Snapshot.Class IN ("Druid", "Hunter", "Shaman", "Warrior")  // Tauren class restriction
Snapshot.Race == "Tauren"                         // starter-zone gate

Snapshot.Position.Zone == "Mulgore"               // current zone signal

Snapshot.QuestLog.Active.BloodhoofQuests          // L5-10 chain progress
```
