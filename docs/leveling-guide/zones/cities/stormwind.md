# Stormwind City — Alliance Capital (Eastern Kingdoms)

Eastern Kingdoms human capital — **the Alliance primary city**. All primary + secondary profession trainers, all class trainers, central Auction House + Bank. **Districts**: Trade (AH/Bank/profession), Old Town (Rogue/Engineering), Mage Quarter (Mage/Warlock + portals), Dwarven District (Hunter), Cathedral Square (Paladin/Priest), The Park (Druid), Stormwind Keep (Highlord Bolvar + King Anduin atmospheric), Stormwind Harbor. **Deeprun Tram** to Ironforge (~3 min ride). Iconic class quest hubs: **Lord Grayson Shadowbreaker** (Paladin Warhorse + Charger chain), **Highlord Bolvar Fordragon** (Onyxia attune chain Alliance turn-in), **Tannysa** (Warlock Felsteed). Mage portals to all Alliance capital cities at L20+ trainer.

## Districts Breakdown

### Trade District (Central AH + Bank + Professions)

**The central hub for trade, banking, and profession crafting**.

| Service | NPC / Location |
|---------|----------------|
| **Auction House** | Central building (cross-faction-tier in goblin ports; Alliance-only in Stormwind) |
| **Bank** | Stormwind Bank (vault + safe-keeper) |
| **All primary profession trainers** | Apprentice through Artisan tiers |
| **All secondary profession trainers** | Cooking + First Aid + Fishing |
| **Specialty trainers (BS Armorsmith → Grumnus Steelshaper, Engineering → Lilith the Lithe nearby)** | Mid-game class+profession specialty unlock |

## Decision-Engine Rules

1. **Cross-Alliance hub**: Stormwind is the **Alliance primary city**. Engine should plan trip for class trainers + profession trainers + AH access at L10+ for all Alliance alts.
2. **Hearthstone bind**: Stormwind Inn is viable for L20-30+ Alliance characters; not bracket-1 default but useful for cross-continent travel.
3. **Class quest dispatch**: per-class district routing (Cathedral Square for Pal/Priest, Mage Quarter for Mage/Warlock, etc.).
4. **Lord Grayson Shadowbreaker priority**: Paladin Warhorse + Charger class chains. Engine should auto-route Paladin alts at L40 + L60.
5. **Onyxia attune Stormwind-centric**: cross-zone Marshal Windsor → Highlord Bolvar chain. Engine should plan multi-step visits.
6. **Tailoring Artisan trainer**: Daryl Stack at Mage Quarter for L20+ Tailoring path.
7. **Mage portal trainer priority**: Teleports at L20 + Portals at L40 unlock cross-continent travel.
8. **Stockades dungeon prep**: L22-30 short Alliance dungeon via Old Town entrance. Engine should plan group queue.
9. **Cross-continent transit hub**: Stormwind → Ironforge tram + Stormwind boats are primary Alliance transit. Engine should cache routes.
