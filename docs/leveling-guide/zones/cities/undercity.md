# Undercity — Horde Forsaken Capital (Beneath Lordaeron Ruins)

Eastern Kingdoms underground capital — **the Horde secondary city**, beneath the ruined Capital City of Lordaeron in Tirisfal Glades. **Districts**: Trade Quarter (AH/Bank — central), Apothecarium (Forsaken-themed Alchemy/Tailoring/LW high-tier), Magic Quarter (Mage/Warlock + portals), War Quarter (Warrior/Hunter), The Rogues' Quarter (Rogue), The Royal Quarter (Sylvanas atmospheric + Varimathras Dreadlord + Royal Apothecary Society), Canals (atmospheric central). **Zeppelin** to Orgrimmar (~3 min ride) + zeppelin to Grom'gol Base Camp (Stranglethorn, after L35 chain). Iconic content: **Sylvanas Windrunner** atmospheric Banshee Queen, **Apothecarium plague-themed Alchemy hub**, Forsaken-themed class quest hubs.

## Decision-Engine Rules

1. **Cross-Horde hub**: Undercity is the **Horde secondary capital** for Forsaken cultural identity. Engine should plan trip for Forsaken alts at L10+ for class trainers + profession trainers + Apothecary path.
2. **Hearthstone bind**: Undercity Inn is preferred for Forsaken alts L10-30. Cross-continent travelers should bind here for fast Tirisfal/Silverpine access.
3. **Apothecarium priority**: Forsaken-themed Alchemy hub (Apothecary Faldren). Engine should auto-route Forsaken alts for Alchemy progression.
4. **Royal Apothecary Society chain**: cross-zone Plaguelands setup. Engine should pickup at L20+ but defer Plaguelands completion to L51+.
5. **Magic Quarter portals**: Teleport: Orgrimmar/UC/Thunder Bluff at L20; Portal versions at L40.
6. **Profession early grind**: Trade Quarter + Apothecarium sufficient for all primary professions (Apprentice through Artisan).
7. **Cross-continent transit hub**: Org→UC zeppelin + UC→Grom'gol zeppelin (after L35) primary Horde transit. Engine should cache routes.
8. **Sylvanas/Varimathras atmospheric**: no quests in 1.12. Engine should NOT plan interaction for quest progression.
