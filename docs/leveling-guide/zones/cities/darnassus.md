# Darnassus — Alliance Night Elf Capital (Atop Teldrassil)

Kalimdor world-tree-top capital — **the Alliance tertiary city** (after Stormwind and Ironforge), built atop Teldrassil. Accessed via **portal from Rut'theran Village** (Teldrassil south coast) or Mage portal at L20+. **Districts**: Tradesmen's Terrace (AH/Bank/profession), Cenarion Enclave (Druid/Hunter — prestigious Druid hub), Temple of the Moon (Priest/Tyrande atmospheric), Warrior's Terrace (Warrior), The Howling Oak (Druid sanctuary), Craftsmen's Terrace (profession). Iconic content: **Cenarion Enclave Druid trainer hub** + **Tyrande Whisperwind** atmospheric NPC + portal cross-zone access. NE-only race-locked trainers in some sub-districts.

**The central hub for trade, banking, and quest hub access**.

### Cenarion Enclave (Druid + Hunter — Prestigious Hub)

**The Druid-class prestige district** — Cenarion-aligned trainers + cross-Cenarion Circle setup.

| Service | NPC / Location |
|---------|----------------|
| **Druid trainers** (e.g., **Loganaar**) | All ranks; class quest origin; **Teleport: Moonglade** spell |

| **Various Cenarion-aligned NPCs** | Cross-Cenarion Circle rep accelerator setup |

**Decision-engine cue:** Cenarion Enclave is **the** Alliance Druid + NE Hunter class trainer hub. Engine should pre-flag NE Druid/Hunter alts.

## Decision-Engine Rules

1. **Cross-Alliance hub**: Darnassus is the **Alliance tertiary capital** for Night Elf cultural identity. Engine should plan trip for NE alts at L10+ for class trainers + profession trainers + Cenarion Enclave Druid/Hunter access.
2. **Hearthstone bind**: Darnassus Inn is preferred for NE alts L10-30. Cross-continent travelers should bind here for fast Teldrassil/Darkshore access.
3. **Cenarion Enclave priority**: NE Druid/Hunter class quest hub. Engine should auto-route NE Druid/Hunter alts at L10.
4. **Druid Teleport: Moonglade**: free 1h-CD spell for Druid alts. Engine should use for Moonglade form quest chains.
5. **Tyrande Whisperwind atmospheric**: no quests in 1.12. Engine should NOT plan Tyrande interaction for quest progression.
6. **Profession early grind**: Tradesmen's + Craftsmen's Terraces sufficient for all primary professions (Apprentice through Artisan).
7. **Cross-continent transit hub**: Rut'theran portal → Auberdine boat → Eastern Kingdoms is primary cross-continent transit for NE alts.
8. **NE racial Starshards Priest**: race-locked. Engine should pre-flag NE Priest alts for Temple of the Moon visit.
