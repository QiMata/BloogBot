# Thunder Bluff — Horde Tauren Capital (Atop Mesa in Central Mulgore)

Kalimdor mesa-top capital — **the Horde tertiary city**, built atop a series of connected rises in central Mulgore. **Districts** (called "Rises"): **High Rise / Trade District** (AH/Bank central — main hub), **Hunter Rise** (Holt Thunderhorn Tauren Hunter trainer hub), **Spirit Rise** (Cairne Bloodhoof atmospheric + Druid trainers + Shaman trainers), **Elder Rise** (additional Druid trainers + Cenarion-aligned NPCs), **Pools of Vision** (Druid sanctuary + atmospheric). Accessed via **ramp from Bloodhoof Village** (Mulgore) — direct. Iconic content: **Holt Thunderhorn** Tauren Hunter trainer + Tame Beast quest origin, **Cairne Bloodhoof** atmospheric High Chieftain, Druid trainers cross-Tauren racial path. **Tauren class restrictions** apply (Druid/Hunter/Shaman/Warrior only — NO Priest/Mage/Warlock/Rogue/Paladin).

## Tauren Class Restrictions

**Tauren can ONLY be**: Druid, Hunter, Shaman, Warrior.

**Tauren CANNOT be**: Priest, Mage, Warlock, Rogue, Paladin.

**Decision-engine rule:** Engine should fail-fast for invalid Tauren-class character creation (e.g., Tauren Mage = invalid). See [../mulgore.md](../mulgore.md) for race-class restrictions detail.

## Decision-Engine Rules

1. **Cross-Horde hub**: Thunder Bluff is the **Horde tertiary capital** for Tauren cultural identity. Engine should plan trip for Tauren alts at L10+ for class trainers + profession trainers + Hunter Rise access.
2. **Hearthstone bind**: Thunder Bluff Inn is preferred for Tauren alts L10-30. Cross-continent travelers should bind here for fast Mulgore/Barrens access.
3. **Tauren class restrictions**: Druid/Hunter/Shaman/Warrior only. Engine should fail-fast for invalid class combos.
4. **Holt Thunderhorn priority**: Tauren Hunter Tame Beast quest origin. Engine should auto-route Tauren Hunter alts at L10.
5. **Cairne Bloodhoof atmospheric**: no quests in 1.12. Engine should NOT plan interaction for quest progression.
6. **Druid Teleport: Moonglade**: free 1h-CD spell for Druid alts. Engine should use for Moonglade form quest chains.
7. **Profession early grind**: High Rise / Trade District sufficient for all primary professions (Apprentice through Artisan).
8. **Cross-continent transit hub**: ramp + Mage portal + Crossroads FP route are primary cross-zone Horde transit. Engine should cache routes.
