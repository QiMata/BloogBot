# Dun Morogh — Dwarf + Gnome Starter Zone (L1-10)

Eastern Kingdoms snowy mountain zone — **Dwarf + Gnome shared starter zone**. Sub-zones: **Coldridge Valley** (Lvl 1-5 Dwarf intro), **Anvilmar** (Lvl 1-5 starter hub), **Kharanos** (Lvl 5-10 main hub), **Steelgrill's Depot** (Gnome refugee area), **Frostmane Hold** (troll cave). Capital adjacent: **Ironforge** (south via tunnel/tram). Notable mobs: **Frostmane Trolls** (cross-zone enemy faction), **Troggs** (Coldridge), **Wendigo**, **Mo'grosh Ogres** (north). Quest hubs lead to **Loch Modan** (L10-15 transition). High Mining density (Copper/Tin nodes prolific).

## Sub-Zone Breakdown

### Coldridge Valley (L1-5 Dwarf Starter)

**Enclosed Dwarf-only starter area** — first 5 levels.

| NPC | Role |
|-----|------|
| **Brewmaster Drohn** | Brewing-themed quest hub |
| **Senir Whitebeard** | Quest giver (paladin/warrior themes) |
| **Dorin Songblade** | Hunter trainer (initial) |
| **Various race trainers** | Class-specific intro |

**Quests (Lvl 1-5):**
- **Senir's Observations** — kill wolves and troggs in Coldridge
- **A Trogg Threat** — kill troggs in Coldridge cave
- **Coldridge Valley Mail** — message delivery to Dun Morogh proper
- **Beer Basted Boar Ribs** — cooking pre-quest

**Decision-engine cue:** Coldridge Valley is the **enclosed L1-5 Dwarf starter**. Engine should plan ~1.5 hour completion through to L5 + Anvilmar exit + Kharanos arrival.

### Anvilmar (L1-5 Hub)

**Coldridge Valley exit point + low-tier services**.

| NPC | Role |
|-----|------|
| **Tannok Frosthammer** | Anvilmar leader |
| **Felix Whindlebolt** | Gnome quest giver |
| **Various trainers** | Class + Mining + Cooking |
| **Innkeeper (low-tier)** | Hearthstone bind option |

**Quests:** transition pre-quest "Mountaineer Stormpike's Task" → Kharanos.

### Kharanos (L5-10 Main Hub)

**The L5-10 quest hub**.

| NPC | Role |
|-----|------|
| **Innkeeper Belm** | Hearthstone bind (Kharanos Inn) |
| **Granis Swiftaxe** | Quest hub leader |
| **Sten Stoutarm** | Quest giver (mountain region) |
| **Tognus Flintfire** | Mining trainer (Apprentice 1-75) |
| **Loslor Rudge** | Engineering trainer (Apprentice 1-75) |

**Quests (L5-10):**
- **Frostmane Threat** — Frostmane troll kill (multiple chains)
- **Wendigo Coats** — kill Wendigos for pelts
- **Trogg Cleanup** — Trogg kills near Kharanos
- **Misty Pine Refuge** — bandit chains south of Kharanos
- **The Public Servant** — Razzeric escort quest (cross-zone Loch Modan setup)

### Steelgrill's Depot (Gnome Refugee Area)

**Eastern hub — Gnome refugees from Gnomeregan**.

| NPC | Role |
|-----|------|

| **Various Gnome trainers** | Engineering (Apprentice + Specialty) |
| **Failure Engineering NPCs** | Gnome-themed quests |

**Quests (L5-12):**
- **Steelgrill's Depot quests** — Engineering and Mining-themed
- **Failed Operations** — Gnome refugee chain (cross-zone Burning Steppes/Searing Gorge eventually)

- **Loslor Rudge**: Engineering (Apprentice 1-75) — Kharanos

### Steelgrill's Depot (Engineering Specialty)

- **Steelgrill** + companions: Engineering trainers (Apprentice through Journeyman)
- Gnome theme — Engineering is bracket-defining for Dwarf/Gnome alts

### Ironforge (Apprentice through Artisan tiers)

- **All primary professions** + specialty trainers
- **Hjalmar Anvilmar** (BS Apprentice + Journeyman)
- **Various Engineering specialty trainers** (Goblin/Gnomish split at lvl 30 + skill 200)
- All secondary professions

**Decision-engine rule:** Kharanos sufficient for L1-15; Steelgrill's Depot for Engineering Apprentice-Journeyman; Ironforge for all advanced professions + specialty.

## Cross-Zone Transition Routes

### To Loch Modan (L10-15 Bracket)

| Path | Difficulty |
|------|-----------|
| South road from Kharanos | Easy (lvl 8-12 mobs); ~5-10 min walk |
| Via Misty Pine Refuge | Direct route through bandit area |

### To Ironforge (L1+)

| Path | Difficulty |
|------|-----------|
| **Tunnel from Kharanos** (south) | Direct; safe |
| Above-ground via Mountain Pass | Slower; mob-encounter risk |

### To Wetlands (L25+ Bracket)

| Path | Difficulty |
|------|-----------|
| North via Mo'grosh Stronghold | Through lvl 25+ mobs; gear-restricted |
| Via Loch Modan → Menethil Harbor → boat | Standard L25+ route |

### To Stormwind (Cross-Capital)

| Path | Difficulty |
|------|-----------|
| Tunnel + Deeprun Tram from Ironforge | Direct; ~3 min ride |
| Via Wetlands + Booty Bay boat (long route) | Multi-zone travel; unnecessary |

## Decision-Engine Rules

1. **Starter zone gate**: Coldridge Valley (L1-5 Dwarf) → Anvilmar → Kharanos (L5-10) → Ironforge (class quests at L10) → Loch Modan (L10-15 transition).
2. **Hearthstone bind**: Kharanos Inn (Innkeeper Belm) is the bracket-1 default. Engine should bind here after first quest from Kharanos.
3. **Class quest dispatch (L10)**: Mage/Hunter/Warlock/Priest/Paladin must travel to Ironforge (Hunter Tame Beast) or Stormwind (Warlock Voidwalker — must travel further). Engine should plan tunnel/tram path.
4. **Loch Modan transition**: at L8-10, head south to Loch Modan via Misty Pine Refuge. Standard route.
5. **Profession early grind**: Kharanos sufficient for L1-15. Steelgrill's Depot for Engineering specialty pre-game. Ironforge for all advanced.
6. **Frostmane Hold party-pull**: Frostmane Chief is elite. Engine should encode group-pull invariant.
7. **Mining grind**: Gol'Bolar Quarry is high Copper/Tin density. Engine should batch Mining + quests.
8. **Cross-tunnel route**: Kharanos→Ironforge via tunnel is the Dwarf/Gnome shortcut to capital. Engine should plan early visit.
