# The Barrens — Horde Mid-Bracket Zone (L10-25)

The largest single-zone bracket in vanilla 1.12.1 — **Horde primary L10-25 zone**, convergence point for all three Horde Kalimdor starter races (Orc/Troll/Tauren). Sub-zones: **The Crossroads** (main hub, highest quest density in vanilla), **Ratchet** (neutral goblin port + boat to Booty Bay), **Camp Taurajo** (Tauren outpost), **Sun Rock Retreat** (south, Stonetalon transition), **Stagnant Oasis** (Wailing Caverns approach + Mankrik's Wife), **Razormane Quilboar Camp**, **Forgotten Pools**, **Bael'dun Keep** (Alliance outpost). **Wailing Caverns** dungeon (lvl 17-24) — bracket-defining 5-man. Iconic Mankrik's Wife meme content. Cross-zone exits: north (Orgrimmar/Durotar), east (Ratchet boat to Booty Bay), south (Sun Rock → Stonetalon Mountains), southwest (Mulgore→Bloodhoof→Thunder Bluff).

## Sub-Zone Breakdown

### The Crossroads (Main Hub)

**The Lvl 10-25 quest hub** — highest quest density in vanilla 1.12.

| NPC | Role |
|-----|------|
| **Sergra Darkthorn** | Main quest hub leader |
| **Mankrik** | "Mankrik's Wife" quest origin (Stagnant Oasis search) |
| **Innkeeper Boorand Plainswind** | Hearthstone bind |
| **Various profession trainers** | Mining + Skinning + Cooking |
| **Stable Master Tiyani** | Hunter pet stable (post-L30 unlock) |
| **Hippogryph Master / Wind Rider Master** | Flight master (FP to Orgrimmar/Camp Taurajo/Ratchet/Thunder Bluff) |

**Quests (L10-25):**
- **The Crossroads quest hub** — 20+ quests in immediate vicinity
- **Mankrik's Wife** — find Mankrik's Wife body at Stagnant Oasis (~10-15 min walk; iconic)
- **Wailing Caverns attune chain** (Naralex) — pre-dungeon quest set
- **Razormane Quilboar quests** — kill quests + reagent collection
- **Centaur faction quests** (Galak/Maraudine/Kolkar — cross-Desolace setup)
- **Wizzlecrank's Shredder** escort quest (multi-stage, ~15 min)
- **Verog the Dervish** rare elite hunt
- **Bael'dun Keep** quest (Alliance dwarves)

### Ratchet (Neutral Goblin Port)

**Goblin coastal port + boat to Booty Bay** (Stranglethorn Vale).

| NPC | Role |
|-----|------|
| **Wharfmaster Dizzywig** | Boat dispatcher |
| **Innkeeper (Ratchet)** | Hearthstone (alt-bind) |
| **Goblin questgivers** | Standard goblin quests |
| **Goblin auctioneer** | Cross-faction AH access |

**Decision-engine cue:** Ratchet boat to Booty Bay is the **fastest cross-continent transition** for Horde (Org → Crossroads → Ratchet → boat). Engine should plan boat-timing for L30+ STV transition.

### Camp Taurajo (Tauren Outpost)

**Tauren-aligned hub south of Crossroads**.

**Quests:** Quilboar kills + Tauren-themed quests + cross-zone Mulgore prep.

## Quest Lines (L10-25)

### Mankrik's Wife (Iconic Meme Quest)

| Step | Action |
|------|--------|
| 1 | Mankrik intro at Crossroads (asks for help) |
| 2 | Travel to **Stagnant Oasis** (~10-15 min walk south + east) |
| 3 | Find Mankrik's Wife body at exact location (~38, 85 coordinates `[verify pass 3]`) |
| 4 | Return to Mankrik for closure |

**Decision-engine cue:** Mankrik's Wife quest is a 1.12.1 meme but mechanically standard. Engine should encode Stagnant Oasis waypoint.

### Wailing Caverns Attune Chain (Disciple of Naralex)

| Step | Action |
|------|--------|
| 1 | Cleo Bonewise / Naralex's NPC line at Crossroads |
| 2 | Multi-quest sub-chain: Glowcap mushroom + Goblin Smoke Powder + various reagents |
| 3 | Free WC quest reward (green item per class) |
| 4 | **Continues into Wailing Caverns dungeon** (Naralex sleeping boss) |

| Step | Action |
|------|--------|
| 1 | Pickup Galak/Maraudine/Kolkar centaur quest at Crossroads |
| 2 | Multi-zone setup (Barrens → Desolace at L28+) |
| 3 | Choose centaur faction: **Magram (Horde-allied) vs Gelkis** (mutually exclusive) |

### Wizzlecrank's Shredder Escort

| Step | Action |
|------|--------|
| 1 | Wizzlecrank in eastern Barrens |
| 2 | Multi-stage escort (~15 min) — Wizzlecrank in shredder mech battling enemies |
| 3 | Final cross-zone setup quest |

## Decision-Engine Rules

1. **Bracket entry**: Barrens is the **L10-25 Horde default zone** + crossroads for all 3 Horde Kalimdor races. Engine should plan ~15-20 hour completion through L20.
2. **Hearthstone bind**: Crossroads Inn is the bracket-2 default. Most flexible bracket-1 hub for Horde.
3. **Mankrik's Wife waypoint**: Stagnant Oasis ~38, 85. Engine should encode meme quest location.
4. **Wailing Caverns attune priority**: pre-quest before dungeon queue. Engine should pickup at L17+.
5. **Verog the Dervish elite**: rare elite hunt; engine should party-pull at L20+.
6. **Centaur faction choice**: mutually exclusive Magram/Gelkis decision affects Desolace L30+ chain. Engine should default to Magram for Horde unless user-config opts out.
7. **Cross-zone Ratchet boat**: fastest cross-continent for Horde (Org → Booty Bay). Engine should plan boat-timing.
8. **Wizzlecrank's Shredder escort**: multi-stage 15-min quest. Engine should plan dedicated time block.
9. **Bael'dun Keep PvP risk**: lvl 20+ Alliance dwarves; engine should defer Bael'dun until L22+.
