# Fishing — 1-300 Secondary Profession, Pool Schools, STV Tournament, Nat Pagle

**Secondary profession** — does NOT count toward 2-primary cap. **No specialty in 1.12.** Pair-feeds **Cooking** + Alchemy reagents (Stonescale Eel, Nightfin Snapper for high-tier potions). Fish tiers: Sagefish (low) → Mithril Head Trout / Glossy Mightfish (mid) → Stonescale Eel / Nightfin Snapper (high). Top events: **Stranglethorn Fishing Extravaganza** every Sunday 14:00-16:00 server time at Booty Bay (rewards: Arcanite Fishin' Pole +35 skill, Hook of the Master Angler trinket, 1000g title). **Nat Pagle** in Dustwallow Marsh sells Fishing books with embedded skill points.

## Skill Progression (1-300)

| Skill | Range | Fish tier | Best zones |
|-------|-------|-----------|------------|
| 1-50 | Apprentice | **Raw Brilliant Smallfish** + **Raw Slitherskin Mackerel** | Capital city pools (Stormwind canals, Org coast, Mulgore lake) |
| 50-100 | Apprentice → Journeyman (75 cap) | **Raw Bristle Whisker Catfish** + **Raw Longjaw Mud Snapper** | Westfall, Loch Modan, Ashenvale water |
| 100-150 | Journeyman | **Raw Rainbow Fin Albacore** + **Raw Mithril Head Trout** | Stranglethorn, Hinterlands, Wetlands |
| 150-225 | Journeyman → Expert (Expert quest at 125) | **Raw Spotted Yellowtail** + **Raw Glossy Mightfish** | Stranglethorn coast, Tanaris coast, Hinterlands |
| 225-260 | Expert → Artisan (Artisan quest at 200) | **Raw Greater Sagefish** + **Raw Sunscale Salmon** | Tanaris, Hinterlands, Hillsbrad |
| 260-300 | Artisan | **Raw Stonescale Eel** + **Raw Nightfin Snapper** + **Raw Whitescale Salmon** + **Raw Redgill** | Tanaris coast, Plaguelands, Hillsbrad |
| Special | All tiers | **Plagued Hatchling Frog** | EPL pools only (skill 280+) |
| Special | All tiers | **Speckled Tastyfish** | Stranglethorn pools (during STV Extravaganza only) |

**Decision-engine rule:** Fishing grind aligns with character level (~Skill = Level × 5). Engine should plan fishing-while-questing routine (cast every 18 seconds at coastal/water pools).

## Pool Schools (Named)

In 1.12, named pools spawn at fixed water locations and **always yield specific fish + ~5-10 catches per pool**. Skill threshold to fish from pools is reduced.

| Pool school | Zone | Skill required | Yields |
|-------------|------|----------------|--------|
| **Stranglethorn Coastal Surface** | STV south coast | 175+ | Raw Glossy Mightfish + Stonescale Eel chance |
| **Floating Wreckage / Debris** | STV (named) | 175+ | Wreckage salvage (5-15g per haul) + Lockboxes (Rogue grind) |
| **Oily Blackmouth School** | Stranglethorn | 130+ | Oily Blackmouth (Alchemy reagent for Free Action Pot) |
| **Sar'theris Striker** | Desolace coast | 175+ | Striker (cooking reagent) |
| **Stonescale Eel Swarm** | Tanaris coast (Steamwheedle Port) | 200+ | Stonescale Eel (Stoneshield Potion reagent) |
| **Nightfin Snapper School** | Felwood / Plaguelands water | 200+ | Nightfin Snapper (Healing Potion reagent) |
| **Plagued Hatchling Frog Swarm** | EPL (Plaguemist Ravine, Plaguewood) | 280+ | Plagued Hatchling Frog (cooking + niche reagent) |
| **Mighty Steamfin Surface** | Stranglethorn (during Extravaganza) | 175+ | Speckled Tastyfish (40-fish goal for Extravaganza) |

**Decision-engine rule:** named pools are time-spawned (15-30 min respawn per pool). Engine should map per-server pool schedule for high-yield grinding routes.

## Stranglethorn Fishing Extravaganza (Sundays)

The **Stranglethorn Fishing Extravaganza** is a weekly tournament at Booty Bay.

| Detail | Value |
|--------|-------|
| Schedule | Sunday 14:00-16:00 **server time** |
| Location | Booty Bay (Stranglethorn Vale) |
| Goal | Catch 40 Speckled Tastyfish from STV pools and deliver to Riggle Bassbait at Booty Bay |
| Win condition | First to deliver = 1st place winner |
| 1st place reward | **1000g** + **Title: "Master Angler of Stranglethorn"** + **Arcanite Fishin' Pole** (+35 skill, BiS) |
| 2nd-4th place | **Hook of the Master Angler** (trinket, +5 skill on use), **Nat Pagle's Extreme Angler FC-5000** (+25 skill), **Lucky Fishing Hat** (+5 skill) |
| Random draws | All other participants get random consolation rewards (cosmetic items, low gold) |

**Decision-engine rule:** STV Extravaganza is **the** Fishing event. Engine should:
1. Track Sunday 14:00-16:00 server time as recurring event
2. Plan STV pool route during 2-hour window
3. Coordinate with raid week-end fishing to avoid conflict
4. Keep Speckled Tastyfish stockpile for next event

## Nat Pagle's Books (Dustwallow)

**Nat Pagle** at **Nat's Landing** in Dustwallow Marsh sells Fishing books that grant skill points + recipes. Each book is a one-time purchase per character.

| Book | Skill bonus | Cost | Notes |
|------|-------------|------|-------|
| **Nat Pagle's Guide to Extreme Anglin'** | +75 fishing skill | 100g | One-time use; instant skill jump |
| **Filet of Redgill recipe** | Cooking recipe | ~10g | Cooking-paired |
| **Mightfish Steak recipe** | Cooking recipe | ~10g | Cooking-paired |
| **Spotted Yellowtail recipe** | Cooking recipe | ~10g | Cooking-paired |

**Decision-engine rule:** Nat Pagle's Guide is **the easy Fishing skill push**. Engine should plan Dustwallow visit at any character with Fishing 200+; pays for itself via reduced grind time.

## Decision-Engine Rules

1. **Fishing is always-on**: secondary profession; engine should default Fishing on every character (no opportunity cost).
2. **Cast cycle**: cast every ~18 seconds at water; auto-loot fish on catch. Engine should encode Fishing cast as standard idle-time activity during AH/raid prep waiting.
3. **Pool priority**: when in range of named pool, prioritize pool over open-water. Pool yields ~5x faster.
4. **Lure usage**: at high-tier pools (skill 250+), auto-apply Aquadynamic Fish Attractor or Bright Baubles. Engine should keep ~30-50 lures stocked.
5. **STV Extravaganza tracking**: Sunday 14:00-16:00 server time recurring event. Engine should mark calendar + plan 2-hour STV pool route.
6. **Nat Pagle's Guide**: at Fishing 200+, purchase Nat Pagle's Guide for +75 skill instant jump. Engine should plan Dustwallow visit.
7. **Stonescale Eel + Nightfin Snapper**: high-tier alchemy reagents. Engine should fish these for raid-prep flask production (Stoneshield + Healing potions).
8. **Plagued Hatchling Frog**: niche EPL pool; only relevant for late-game cooking + 1.12-specific lore content.
9. **Cooking + Fishing pair**: secondary professions; engine should grind both in parallel for self-sustaining fish-tier cooking.
10. **AH-flipping fish**: at L60, Stonescale Eel + Nightfin Snapper are 5-15s per fish on AH. Engine should AH-flip when stockpile exceeds personal alchemy need.
11. **Pole + lure stacking**: total effective skill = `Profession.Fishing.Skill + Pole.Bonus + Lure.Bonus`. Engine should compute total before attempting high-tier pools.
