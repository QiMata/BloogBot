# Herbalism — 1-300 Gathering, Herb Tiers, Black Lotus Hunt

Pure gathering profession. **No specialty in 1.12.** Pair-feeds **Alchemy** (almost exclusively). Herb tiers: Peacebloom/Silverleaf/Earthroot (lvl 1) → Mageroyal/Briarthorn (mid-low) → Liferoot/Khadgar's Whisker (mid) → Sungrass/Blindweed (high) → Dreamfoil/Mountain Silversage/Plaguebloom/Icecap (max-tier) → **Black Lotus** (rare elite-zone spawn — 5-tier flask reagent). Top zones: Felwood / Plaguelands / Silithus / Winterspring / Searing Gorge / Burning Steppes. **Black Lotus is the most contested herb in 1.12** — required for raid-tier flasks (Mongoose, Titans, Wisdom, Supreme Power, Distilled Wisdom).

## Black Lotus Hunt (1.12 Most-Contested Resource)

Black Lotus has **5 spawn zones** in 1.12.1, each with **~5-7 fixed spawn nodes** + **4-12 hour respawn timer per node** (varies by server). Total server-wide spawns: ~25-35 active at any time.

| Zone | Spawn nodes | Notes |
|------|-------------|-------|
| **Burning Steppes** | ~6 nodes (Blackrock Mountain perimeter, Dreadmaul Hold, Altar of Storms) | Heavy contested (raid attune zone) |
| **Eastern Plaguelands** | ~5 nodes (Tyr's Hand area, Plaguewood, Stratholme exterior) | Heavy contested |
| **Western Plaguelands** | ~5 nodes (Andorhal, Hearthglen, Sorrow Hill) | Medium contested |
| **Silithus** | ~7 nodes (Twilight camps, Hive perimeter) | Medium contested |
| **Winterspring** | ~5 nodes (Frostsaber Rock, Mazthoril, Demon Fall Canyon) | Light contested (high-level mobs nearby) |

**Each Black Lotus = 1 raid-tier flask**:
- Flask of the Titans (8 health, +400 HP)
- Flask of Distilled Wisdom (+65 Spirit / +2000 mana)
- Flask of Supreme Power (+150 Spell Damage)
- Flask of Mongoose (+25 Agility / +2% crit)
- Flask of Chromatic Resistance (+25 all res)

**Decision-engine rule:** Black Lotus is **server-economy-defining** — typical AH price 30-100g per Black Lotus, and contested farming is a significant time-investment. Engine should:
1. Map Black Lotus spawn nodes per server zone
2. Track respawn timers (4-12h per node)
3. Prioritize Black Lotus over other herbs when in range of a spawn point
4. Coordinate raid-flask production with raid leaders for guild-locked flasks

## Cenarion Circle Reagent Donation

Herbalists can turn in herbs at Cenarion Hold for Cenarion Circle rep + Cenarion Reservist's pouch:

| Herb pouch | Skill required | Rep gain | Notes |
|------------|---------------|----------|-------|
| **Cenarion Reagents** | Herbalism 200+ | ~75-150 rep per pouch | Various herb-tier-mixed donations |
| **Cenarion Cleansing Brew** | Alchemy 250+ + Herbalism material | ~250 rep per brew | Alchemy partner; CC rep accelerator |
| **Twilight Cultist Mantle** + herb donation | Cenarion Hold quest line | ~500 rep per turn-in | Twilight's Hammer cultist drops + herb mix |

**Decision-engine rule:** Cenarion Circle Honored is reachable via ~30-50 Herbalism donations + Silithus quests. Engine should batch herb donations during dungeon downtime.

## Herbalism + Alchemy Pair Strategy

For pair-locked profession choice:

| Character class | Recommended pair |
|-----------------|------------------|
| Any caster (Mage / Priest / Warlock / Druid / Shaman caster) | Herbalism + Alchemy (flask production) |
| Druid (any spec) | Herbalism + Alchemy (Travel Form = fastest herb route) |
| Hunter | Mining + Engineering preferred; Herbalism + Alchemy as alt-pair |
| Tank/Melee classes | Mining + Blacksmithing preferred; Alchemy as alt for mid-game potions |
| Solo flask self-sustain | Herbalism + Alchemy on same character |

**Decision-engine rule:** Druid + Travel Form + Herbalism is **optimal solo herb-grind combo**. Engine should bias Druid alts toward Herbalism+Alchemy pair for L60 endgame flask self-sustain.

## Decision-Engine Rules

1. **Profession pair-lock**: Herbalism feeds Alchemy. Engine should default Herbalism on Druid/Mage/Priest/Warlock/Shaman (caster) characters with Alchemy partner.
2. **Herb-tier-to-zone routing**: engine should plan gathering loops by zone:
   - Felwood: Sungrass + Blindweed + Gromsblood + Golden Sansam + Dreamfoil + Mountain Silversage (densest single zone)
   - Silithus: Mountain Silversage + Dreamfoil + Black Lotus
   - Winterspring: Icecap + Mountain Silversage + Black Lotus
   - Plaguelands: Plaguebloom + Arthas' Tears + Mountain Silversage
3. **Find Herbs always-active**: minimap herb radar; engine should check `Spells.Has("Find Herbs")` and toggle on for Herbalism characters.
4. **Black Lotus priority**: server-wide contested resource. Engine should map nodes per zone + track respawn timers + prioritize Black Lotus over other herbs in range.
5. **Cenarion Circle donation parallel**: herb-stockpile donations at Cenarion Hold for CC rep. Engine should coordinate flask production with raid-flask need vs. donation rep accelerator.
6. **Find Herbs vs Track Plants**: in 1.12, `Find Herbs` is the trained ability; `Find Plants` is a TBC term. Engine should use 1.12-correct spell name.
7. **Druid + Travel Form bias**: Druid herbalists with Travel Form skill (lvl 30+) double herb-loop efficiency vs other classes. Engine should pre-flag Druid+Herbalism for endgame flask production.
8. **Herb-AH flipping at L60**: Black Lotus 30-100g/each AH; Mountain Silversage 50-80s; Dreamfoil 40-60s. Engine should AH-flip when alchemist alt has full flask reagents.
9. **Cave Ghost Mushroom**: Maraudon caves, Hinterlands caves, Tanaris underground. Engine should batch Ghost Mushroom collection during dungeon-zone clears.
10. **Wintersbite Alterac-only**: 1.12-specific zone gate. Engine should plan Alterac Mountains visit at L40+ for Wintersbite collection (used in mid-tier potions).
11. **Cooking pair-stub**: some Herbalism reagents (Sungrass, Mountain Silversage) feed Cooking-tier recipes. Engine should reserve 5-10% of high-tier herb pile for Cooking partner.
