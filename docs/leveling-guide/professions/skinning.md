# Skinning — 1-300 Gathering, Leather Tiers, Devilsaur + Dragonscale Farming

Pure gathering profession. **No specialty in 1.12.** Pair-feeds **Leatherworking** (~99% of material flow). Leather tiers: Light → Medium → Heavy → Thick → Rugged → **Heavy Devilsaur Leather** (Un'Goro elites) → **Black Dragonscale** (BWL trash) → **Onyxia Scale** (Onyxia drop) → **Pristine Hide of the Beast** (Druid quest / ST drop). Top farming zones: Un'Goro (Devilsaurs), Stranglethorn (cats/raptors), Winterspring (Frostsabers), Plaguelands (Plagued bears), Felwood (corrupted beasts).

## Skill Progression (1-300)

| Skill | Range | Beast level | Leather drop |
|-------|-------|-------------|--------------|
| 1-50 | Apprentice | Lvl 1-15 beasts (wolves, boars, bears, cats in starter zones) | **Light Leather**, **Light Hide** (rare) |
| 50-100 | Apprentice → Journeyman (75 cap) | Lvl 16-25 beasts (gnolls, tigers, kodos, raptors in mid-low zones) | **Medium Leather**, **Medium Hide** |
| 100-150 | Journeyman | Lvl 25-35 beasts (panthers, wolves, bears in mid zones) | **Heavy Leather**, **Heavy Hide** |
| 150-225 | Journeyman → Expert (Expert quest at 125) | Lvl 35-50 beasts (cats in STV, raptors in Hinterlands, hyenas in Tanaris) | **Thick Leather**, **Thick Hide** |
| 225-260 | Expert → Artisan (Artisan quest at 200) | Lvl 50-60 beasts (Plaguelands bears, Felwood corrupted) | **Rugged Leather**, **Rugged Hide** |
| 260-300 | Artisan | Lvl 55-60 elite beasts (Devilsaurs, Stegodons, dragonkin) | **Rugged Leather + special drops** (Heavy Devilsaur, Black Dragonscale, etc.) |

**Decision-engine rule:** Skinning grind aligns with character level (~Skill = Level × 5). Engine should plan kill-route through skin-rich zones based on bracket.

## Special Skin Sources

### Heavy Devilsaur Leather (Un'Goro)

**Source:** Devilsaurs in Un'Goro Crater (lvl 55 elite, ~6-8 named spawn points + roaming patrols).

**Difficulty:** Devilsaurs are **soloable at L60+ for plate/feral classes**, otherwise pair/group required. Hunter solo-friendly with kiting (Disengage at 30%, Frost Trap on cooldown).

**Drop rate:** ~80-100% per Devilsaur kill (1-2 Heavy Devilsaur Leather per skin) + ~5-10% chance for **Devilsaur Eye** (rare reagent).

**AH price:** 50-100g per Heavy Devilsaur Leather (1.12-server-dependent); single Devilsaur Tunic recipe needs 10 Heavy Devilsaur Leather.

**Decision-engine rule:** Un'Goro Devilsaur farming is **the** AH-flipping path for Skinners. Engine should farm Devilsaurs daily (4-6 hour respawn timer per spawn point) for L55+ Skinner alts.

### Black Dragonscale (BWL)

**Source:** BWL trash drops (Drakonid + Black Drakes) and Nefarian. Each BWL clear yields ~5-10 Black Dragonscale on average.

**AH price:** 30-50g per scale.

**Use:** Black Dragonscale Boots / Shoulders / Breastplate (Dragonscale LW recipes); top-tier mail.

### Onyxia Scale

**Source:** Onyxia raid drop (kill yields ~6-10 Onyxia Scale per raid).

**Use:** Onyxia Scale Cloak (33% Onyxia/Nefarian fire breath resist) — universal LW recipe.

### Pristine Hide of the Beast

**Source:** Druid class quest reward (Pristine Hide of the Beast) OR Sunken Temple boss drop (rare).

**Use:** Devilsaur set + Hide of the Wild + Robe of the Archmage + Truefaith Vestments.

**Decision-engine rule:** Pristine Hide is **bottleneck reagent** for multiple class-epic recipes. Engine should always pickup Druid Pristine Hide quest at L50; ST farm runs as backup.

### Brilliant Chromatic Scale (BWL Chromaggus)

**Source:** Chromaggus boss in BWL.

**Use:** Cloak of Brilliance + niche endgame LW recipes.

## Skinning + Leatherworking Pair Strategy

For pair-locked profession choice:

| Character class | Recommended pair |
|-----------------|------------------|
| Hunter | Skinning + LW (Dragonscale spec, Devilsaur set) — natural pair |
| Druid | Skinning + LW (Tribal spec, Wolfshead) — natural pair |
| Rogue | Skinning + LW (Dragonscale or Tribal) — DPS/stealth gear pair |
| Shaman Enh | Skinning + LW (Dragonscale, Stormshroud) |
| Warrior tank/DPS | Mining + BS preferred; Skinning + LW alt-pair for Onyxia Scale Cloak |
| Caster (Mage/Priest/Warlock) | Skinning + LW only if pursuing Robe of the Archmage / Truefaith Vestments via cross-prof; otherwise Tailoring solo |

**Decision-engine rule:** Skinning + LW is **the natural pair** for Hunter/Druid/Rogue. Engine should default these alts to Skinning + LW.

## Decision-Engine Rules

1. **Profession pair-lock**: Skinning feeds LW. Engine should default Skinning on Hunter/Druid/Rogue/Shaman characters with LW partner.
2. **Beast-tier-to-zone routing**: engine should plan kill-loops through skin-rich zones:
   - L1-30: starter + Westfall/Loch Modan/Barrens
   - L30-50: Stranglethorn (Hemet expedition cats/raptors)
   - L50-58: Plaguelands + Felwood + Winterspring + Tanaris
   - L55-60: **Un'Goro Devilsaurs** (Heavy Devilsaur Leather farm)
3. **Zulian Slicer equip**: at L60 with Skinning 285+, engine should auto-equip Zulian Slicer when farming routes are active.
4. **Devilsaur farming**: 4-6 hour respawn timer per spawn point. Engine should track per-server respawn calendar.
5. **Black Dragonscale BWL accumulation**: long-term reagent for Black Dragonscale set. Engine should track BWL trash drops.
6. **Onyxia Scale post-raid loot**: reagent for Onyxia Scale Cloak. Engine should DKP/loot-priority for Onyxia Scale on Onyxia kills.
7. **Pristine Hide quest priority**: at L50, Druid alts should take Pristine Hide quest. Other classes farm at Sunken Temple.
8. **AH-flipping rate**: Heavy Devilsaur Leather + Black Dragonscale are the highest-margin AH flips. Engine should AH-flip at L60 if LW partner doesn't need supply.
9. **Cenarion Circle herb donation parallel**: Skinning has no rep accelerator; engine should focus on raw skill progression.
10. **Tool slot management**: at L60, Zulian Slicer (+25 skill) is bracket-defining; pre-Zulian, Finkle's Skinner (+5) is mid-tier baseline.
