# Leatherworking — 1-300 Grind, 3-Way Specialty, Devilsaur & Wolfshead

The leather + mail crafting profession. Pair-locks with **Skinning**. **3-way specialty at lvl 40 + skill 225**: **Dragonscale** (agi mail/leather DPS, Hunter/Shaman Enh) vs **Tribal** (Druid form-helm, caster leather) vs **Elemental** (FR set, caster mail). Specialty change is permanent in 1.12 (un-learn + re-grind to swap). Top recipes: **Devilsaur Leggings + Tunic** (Hunter BiS pre-raid, Dragonscale), **Wolfshead Helm** (Druid feral BiS, Tribal), **Onyxia Scale Cloak** (33% fire-breath reduction, all LW), **Volcanic set** (FR for MC, Elemental), **Stormshroud Armor** (Shaman Enh BiS pre-raid).

| Class synergy | Hunter (Dragonscale primary), Druid (Tribal primary), Shaman/Rogue (Dragonscale secondary), Resto Druid (Tribal/Elemental) |

| 100-150 | Journeyman | Heavy Leather (lvl 25-35 beasts); Cured Heavy Hide; Hillman's set | Same |
| 150-200 | Journeyman → Expert (Expert quest at 125) | Thick Leather (lvl 35-50 beasts); Dusky Leather Bag (10-slot); Wolfshead Helm precursor `[verify pass 3]` | Same |

**Decision-engine rule:** LW grind aligns with Skinning grind (`Skill.LW <= Skill.Skinning`). Engine should pause LW if Skinning is too far behind; pair character class to match (Hunter/Druid/Rogue typical for LW).

## Decision-Engine Rules

1. **Pair-lock with Skinning**: `Profession.Skinning.Skill >= Profession.Leatherworking.Skill - 50`. Engine should pause LW if Skinning lags.
2. **Specialty selection at lvl 40 + skill 225**: prompt user-config (`Snapshot.Profession.PreferredSpecialty`):
   - Hunter / Shaman Enh → **Dragonscale** (Devilsaur set, Stormshroud)
   - Druid (any spec) → **Tribal** (Wolfshead, Hide of the Wild)
   - Tank or FR-MC raider → **Elemental** (Volcanic set)
3. **Specialty change protocol**: requires un-learning + re-leveling 1-300 + re-traveling to new specialty trainer; engine should refuse-by-default; require explicit user-confirm flag.
4. **Wolfshead Helm priority**: Druid Feral BiS pre-raid; engine should auto-craft for any Druid alt at L50+ with Tribal spec.
5. **Devilsaur 2-piece for Hunter**: BiS pre-raid leg + chest. Engine should grind Heavy Devilsaur Leather in Un'Goro at L55+.
6. **Onyxia Scale Cloak baseline**: at L60 with LW 300, engine should auto-craft for any tank/melee character before MC entry.
7. **Cured Rugged Hide cross-stockpile**: LW alt should produce Cured Rugged Hide for Tailoring partner (Mage/Priest epic chest reagents).
8. **Heavy Devilsaur Leather farming**: Un'Goro Devilsaurs (lvl 55 elites; pair-pull at lvl 58+ solo). Engine should run Devilsaur farming during L55-58 questing in Un'Goro.
9. **Volcanic set for MC tanks**: Elemental spec is **the only path to Volcanic FR set**. Engine should pre-flag Elemental for plate-tank Warriors with FR-need (~MC raider).
10. **Trainer-zone alignment**: Dragonscale (Azshara/Badlands), Tribal (Feralas/STV), Elemental (Searing Gorge/Arathi). Engine should schedule trainer-visit during corresponding bracket questing.
11. **Reagent stockpile (Heart of Fire, Pristine Hide, Black Dragonscale)**: long-term collection during raid weeks. Engine should track per-recipe reagent need.
